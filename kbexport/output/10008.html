Changing one-touch on SOAP

10008

Introduction

Changing what happens when you click on a one-touch icon - the icon right to the left of a description of an assessment or treatment on the SOAP screen

Symptom

Cause

The service is set by what is in the corresponding menu

Workaround

Resolution

Go to Configuration | Menus | Default One Touch for Assessments or Default One Touch for Treatments / (treatment type) and create a new treatment in the menu

Notes
