One Touch NKA

10029

Introduction

Symptom

One Touch NKA is not closing the Allergies window.

Cause

The Close Window flag is not set on that menu button.

Workaround

Resolution

Go to the allergy screen in config mode and click on the black menu text.

Create a local copy, if necessary, and edit the menu.

Select the One Touch NKA button and then click on the Edit button.

Int he top right of the window is a box labeled “Auto Close Window” that says “No.”

Click on the No and select “Finished.”

Click Finished, OK, and Finished.
