Unable to create Version 2 Encounters - License Error

10031

Introduction

This article addresses not being able to open or log in to EncounterPRO interface due to a license error.

Symptom

A screen containing an error message like this:

  1. Happening on all computers.
  2. Happening on one or some computers.

Cause

  1. Happening on all computers - Improperly licensed user. (Only applies to EncounterPRO version2)

    • A new physician or extender (NP, PA, etc) was recently added without updating the JMJ license file on the server.

    • License server needs to be restarted

    • If this does not fix the problem. Follow instructions in below KB article

    • https://192.168.155.2/kb/Article.aspx?ArticleNumber=10284

  2. Happening on one or some computers - Incorrect clock settings. (Applies to all verisons of the EncounterPRO program)
    • Settings of the clock on the server do not sufficiently match those of the computer getting this error

Workaround

Resolution

1- Improperly licensed user -

Contact JMJ Support to create a new license file

2- Incorrect clock settings -

Open Date/Time settings on this computer. Compare the time, date, year, time zone and Daylight savings settings to those of the server. Adjust accordingly.
