Adding Print button to Rx Dashboard

Introduction

Symptom

Cause

Workaround

Resolution

Go to Configuration, Treatment Type, Prescription Drugs, Edit, Services tab, Add Service, pick EncounterPRO Report.

Find that EncounterPRO  report service at the bottom of the list. Click on it and select Edit Service. Change the top flag to “Yes”.

Change the button title to “Print Rx”; change the Button to button 33.bmp. Click OK.

Select the service again; click Configure Service; Leave “Select Destination” blank, select report, treatment, drug prescription; all other fields are left blank.

Move the new service to the desired location (usually up).

Finish out of configuration process and check dashboard for button and its function.
