Can't change user color

10042

Introduction

This is an article about not being able to change user colors in Configuration, Users

Symptom

Going to Config, Users, clicking on a user, selecting the color bar, getting an error like this:

Error calling external object function select_color at line 3 in clicked event of object st_color of w_user_definition.

where   :w_user_definition
object  :st_color
event   :clicked
line    :3

Cause

A corrupted ocx registration

Workaround

Resolution

For each computer experiencing the problem, go to a command prompt, navigate to either Winnt or Windows, then system32, then run this command:

Version 2:

regsvr32 axservic.ocx cicink.ocx easyink.ocx comdlg32.ocx mmpanel.dll

Version 4: (Pre 4.05.35 ONLY)

regsvr32 axservic.ocx cicink.ocx easyink.ocx comdlg32.ocx

If that doesn't work, then install the latest standalone EproLib4 component.

If that doesn't work, then uninstall/reinstall EncounterPRO
