How to edit the Current JMJ Owner field on the Outlook Issue form

10046

Introduction

This article contains detailed instructions for editing the list of issue owners on JMJ's Issue form used in MS Outlook.

Symptom


Cause


Workaround


Resolution

  1. Browse to the Issues folder in Outlook:


  2. Go to Tools\Forms\Design Form...


  3. Look in Outlook Folders and open the form named “Issues“


  4. Delete the Issue Subject


  5. Go to the All Fields tab and delete the values for Issue ID, Issue Number, Issue Status Date, and Owner Start Date


  6. Go to the Issue Status tab, right-click the Current JMJ Owner combo-box control and select Properties


  7. On the Value tab, edit the Possible Values field, then click OK to close the dialog


  8. To publish the form to the Issues Outlook folder, click thePublish Form button on the top left corner of the toolbar, or selectTools\Forms\Publish Form...


  9. Close the form editing window and do not save changes when prompted.
  10. Right-click issues in the Outlook browser and select Properties


  11. On the Forms tab, click Manage.. to open the Forms Manager


  12. Select the issues form in the right-side column and click Copy.  Click 'Yes' on any dialogs.
  13. Select the issues form in the left-side column and click Copy.  Click 'Yes' on any dialogs.
  14. Click Clear Cache

Notes

To perform these operations you need the proper privileges on the Issues folder and the Issues form within.


Changes you make may be lost if you add an issue owner without following the procedures located here:
