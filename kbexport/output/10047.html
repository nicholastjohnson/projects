Changing the prescription size

 To change the prescription size from 2 signature line to 1signature line or vice versa, change all the atttribute inc_report_attribute for attribute name = "dataobject".

2 Signature line to 1 signature line
report_idattribute_sequenceattributevalue
{ED21F61C-D7A2-40EF-9022-2FE20658839D}23dataobjectdw_prescription_size_20

1 Signature line to 2 signature line
report_idattribute_sequenceattributevalue
{ED21F61C-D7A2-40EF-9022-2FE20658839D}23dataobjectdw_prescription2_size_20

Note the use of the %SIGNING_PROVIDER_LICENSE%token.  EncounterPRO will replace this token at runtime with thename and license number of the provider who owns or ordered theprescription.  Other supported tokens for this field are:

Token

Substitution Value

Example

%SIGNING_PROVIDER%

Name of prescription owner

Familia, George K. MD

%SIGNING_PROVIDER_LICENSE%

Name and license number of prescription owner

Familia, George K. MD, Lic# 1234567

%SIGNING_PROVIDER_NPI%

Name and NPI of prescription owner

Familia, George K. MD, NPI: 9876543

%SIGNING_PROVIDER_LICENSE_NPI%

Name, license number, and NPI of prescription owner

Familia, George K. MD, Lic#1234567, NPI: 9876543

The NPI tokens require EncounterPRO Client 35.2 (or later) and EncounterPRO Server 6.1.2 (or later)

Changing Font

Add the following two records to c_report_attribute, EncounterPRO defaults to Arial Narrow


report_idattribute_sequenceattributevalue

{AD78CA36-789B-49F1-8C17-D76C730E488F}360dwmodify_fontsignature.Font.Face='Arial'

Moving the Patient Name/Address and Drug Prescription up or down

Add the following records inc_report_attribute to move everything below the upper black line. Make the number larger than the default to move everything down. Make the number smaller than the default to move everything up. See table below for default values.


report_idattribute_sequenceattributevalue
{ED21F61C-D7A2-40EF-9022-2FE20658839D}359dwmodify_pt_addressdatawindow.header.height=2208   
{AD78CA36-789B-49F1-8C17-D76C730E488F}361dwmodify_pt_addressdatawindow.header.height=2208   

Changing the left and right printer margins to allow more or less room on the printed page

Add the following records inc_report_attribute to change the left and right print margins to 1/4inch.  Increase the values to increase the margins (shrinking theprintable area and forcing more wrapping).  If you set thesevalues lower than the minimum allowed for the printer, most printerswill simply ignore the value and use their own minimums.


report_idattribute_sequenceattributevalue
{ED21F61C-D7A2-40EF-9022-2FE20658839D}359dwmodify_print_lmdatawindow.print.margin.left=250   
{AD78CA36-789B-49F1-8C17-D76C730E488F}361dwmodify_print_lmdatawindow.print.margin.left=250
{ED21F61C-D7A2-40EF-9022-2FE20658839D}359dwmodify_print_rmdatawindow.print.margin.right=250 
{AD78CA36-789B-49F1-8C17-D76C730E488F}361dwmodify_print_rmdatawindow.print.margin.right=250
