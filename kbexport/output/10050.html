Missing Todo Item

10050

Introduction

A Todo Item is temporarily missing from the user's Todo List.

Symptom

When the user performs a todo item and clicks “I'll Be Back”, the todo item disappears from the user's todo list.  A minute later, the todo item reappears.

Cause

This problem is caused by logic in the database which suppresses todo items which have been started within the last minute.  This logic prevents runaway loops in the background server services which process todo items using the same queueing logic as the users todo item screen.

Workaround

The only workaround is for the user to wait one minute for the item to reappear.  A permanent solution however is to apply Hotfix 24 or upgrade to Database modification level 121.

Resolution

Apply Hotfix 24 or upgrade to Database modification level 121.
