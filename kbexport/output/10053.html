Staying logged in to EncounterPRO v2

10053

Introduction

This is an easy copy-paste explanation for customers on how to set a computer to secure in EncounterPRO, version 2.

Symptom

Cause

Workaround

Resolution

To get the Stay Logged On option when logging in to EncounterPRO, go to Configuration, Preferences and set the Level=System and with the Preference Type set to General, click on New Preference, then Computer Secure, then enter in "true" (without the quotes). Then log off and close EncounterPRO. Next time you open EncounterPRO on this computer, the Stay Logged On option should appear.
 
NOTE: This will have to be done on each computer where you want this option to appear (unless this computer is a thin client). Also, this will make the contents of the Office screen appear before logging on.
