Immunization procedure not configured with corresponding vaccine

10055

Introduction

This explains in customer friendly language how to fix this error.

Symptom

Cause

If the error happens when ordering the vaccine from anywhere, then the problem is caused by the Vaccine not having an associated vaccine procedure.  If the problem only happens when ordering the vaccine from the Vaccine Tab/View Detail screen, then the problem is caused by the Immunization treatment incorrectly having a Vaccine Admin service configured as a Past-Treatment-Autoperform service.

Workaround

Resolution

If the problem happens when ordering the vaccine from any screen, then do the following to fix it:

1.  Go to Configuration, Procedures, set Procedure Type = Vaccines, find the vaccine (search for it thru Description if necessary and set Specialty to All), click on the procedure and then Edit Procedure.

2.  Click on the blank bar next to "Vaccine: " and then select the corresponding vaccine. Then click OK to apply the changes. This should take effect for all pts in future visits and any pts currently in the office experiencing this problem with this vaccine.

If the problem only happens when the vaccine is ordered from the Vaccine Tab/View Detail screen, then fix it by doing the following:

1.  Go to Configuration, Treatment Types, find the Immunization treatment type, click on it and then click Edit Treatment

2.  Click on the Services tab.  Find the service where the "Past" column is "Yes".  Click on that service and select "Remove Service".
