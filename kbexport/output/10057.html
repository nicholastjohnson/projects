Remove DEA # from prescriptions

10057

Introduction

Symptom

Cause

Workaround

Resolution

To remove the DEA #  from prescriptions printing out, this can be done by going to Config, Reports and
1- Under the Treatment tab, select Drug Prescription, Default, and set the answer on the first screen to No
(This may need to be repeated "Per Office" for each office if the practice has multiple offices.)
 
2- Under the Encounter tab, select Prescrition, Default, and set the answer on the first screen to No
(Same note under #1 applies here also.)

Notes

This may require restarting the EncounterPRO Report serivce when done with above.
