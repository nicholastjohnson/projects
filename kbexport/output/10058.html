Setting up and features of EPRO SERVER

10058

Introduction

This article provides information on how to install EPRO Server, it's features and explanation of DB Maintenance.

Symptom

Cause

Beginning with EncounterPRO 4.05.XX, the EncounterPRO Server capability has been broken out from the EncounterPRO client.  The EncounterPRO server services have been consolidated into a single service called “EncounterPRO Service”.  This service runs a program called EncounterPROService.exe which reads the o_Server_Components table and spawns the appropriate threads to run all the server functions.

The old server services are disabled for the time being to allow a quick return to the old configuration should problems arise with the new EncounterPRO Service.  These services will eventually be deleted.

Workaround

Resolution

Step # 1 - Installing EPRO Server

  1. Download EPRO Server 2.8 from FTP \EP_Upgrades\Production\4.05\EncounterPRO Server\2.8
  2. Stop all EPRO Services
  3. If previous version of EPRO Server has been installed it will need to be removed
  4. All sessions of Encounterpro.exe and eproserver.exe need to be ended
  5. Run Setup.exe
  6. Select “Next”
  7. Select “Install Services" Usually the password for the JMJ Service account is !$Epro&&
  8. Select “Next”
  9. Select "Copy…" box
  10. Select “Next”
  11. It will usually default to the Data Drive. If not point it to the same drive as where EPRO is installed. For example "D:\Program Files\JMJ\EncounterPRO Server"
  12. Select “Next”
  13. Select “Next”
  14. Select “Install”
  15. Select “Finish”
  16. Upgrade Mod Level to 126
  17. Install all Hot Fixes
  18. Open up Services and Enable the EncounterPRO Service and Disable old services
  19. Check o_server_component
  20. Start EncounterPRO Service
  21. Ensure that EncounterPRO opens and that the Calendar works and they you can view attachments. If not you will need to install latest EPROLIBNET from FTP

 Step # 2 - Explanation of the Tabs in EPRO Server

In EncounterPRO Server we got rid of a lot of tabs.  The only ones remaining are as follows:

  1. General – View/set SQL login, db security mode, remote server, database status, and beta status.
  2. Updates – Manage updates to EncounterPRO systems
  3. Migrate – Manage migration from version 2 to version 4
  4. Hotfix Scripts – Display/Execute hotfix scripts available for current database mod level.  Normally you want run all Hotfix Scripts.  The screen indicates which ones have already been run.
  5. Utility Scripts – Display/Execute utility scripts available for current database mod level

Step # 3 - Features of EPRO Server

  1. There is now a separate InstallShield Setup program for EncounterPRO Server.  It installs a program called eproserver.exe into a directory separate from the client installation.  You can find the new set program on the CD under EncounterPRO\Server.  This program should not normally be installed on fat clients.
  2. The ask screen of the client no longer allows “Server” or “DB Maintenance” options.  Clicking “OK” always starts the client program.
  3. To use the DB Maintenance screens, run eproserver.exe.  A shortcut to eproserver.exe should be put into the JMJ Tools folder.
