Prescription Report Paper Management

10060

Introduction

The prescription report has a number or attributes which can be used to manage the printer and paper used to print the prescriptions.

Symptom

Cause

Workaround

Resolution

Attribute:          dataobject

Definition:         A string designating which powerbuilder datawindow to use

Values:             dw_prescription                        Print area width = 3.75”,  Footer = 6.354”

                        dw_prescription_size_6            Print area width = 5”,  Footer = 1.083”

                        dw_prescription_size_20          Print area width = 3.5”,  Footer = 4.864”

                        dw_prescription2                      2 signature line version

                        dw_prescription2_size_6          2 signature line version

                        dw_prescription2_size_20        2 signature line version

Attribute:          datawindow.print.paper.source

Definition:         An integer indicating which printer tray to pull paper from

Values:             0 — Default

1 — Upper

2 — Lower

3 — Middle

4 — Manual

5 — Envelope

6 — Envelope manual

7 — Auto

8 — Tractor

9 — Smallfmt

10 — Largefmt

11 — Large capacity

14 — Cassette

 

 

Attribute:          datawindow.footer.height

Definition:         An integer indicating how tall the footer section is in thousandth of an inch.  The default footer height depends on the “dataobject” attribute (see above).  Increasing the footer height has the effect of raising the signature section on the page.  Decreasing the footer height has the effect of lowering the signature section on the page.
