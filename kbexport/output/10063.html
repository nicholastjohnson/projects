Sending bills to the primary provider instead of supervisor account

10063

Introduction

Sometimes users requested to send bills to patients primary providers instead of the NP/PA supervisors. There are only 2 ways we can handle NP/PA encounters.

a) Set the external (PMS) billing id for the NP/PA in c_user table. Assign a supervisor for the PA/NP user record. This always bills PA/NP encounters under their superviosr account

b) Set the external (PMS) billing id for the NP/PA in b_resource table. This actually bills their encounters always under the patient primary provider.
