ERROR Facility Code not mapped

10064

Introduction

This is an article discussing how to trouble shoot an error about a facility code.

Symptom

Some but not necessarily all patients are have difficulty checking in thru the practice management system.

 

A message like this in Event Viewer:

Event Type: Error

Event Source: EncounterPROTransportIn

4.05.33.7 (10/8/2004) >>> u_hl7_arrived_mpm - (arrived_mpm()) Facility Code (1006) not mapped.

Cause

Unmapped facility code

Workaround

Practice will have to check these patients in manually until this is fixed.

Resolution

Change the billing_id for the practice accordingly. Also add the facility code in c_component_attribute under

ENCOUNTERPRO_SCHED.facilityid

Notes

See also support issue 16175
