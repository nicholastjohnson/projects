EPImageViewer cannot open an attachment

10070

Introduction


Symptom

  • You get an error similar to the one below:

    System.OutOfMemoryException: Out of memory.
       at System.Drawing.Image.FromFile(String filename, Boolean useEmbeddedColorManagement)
       at EPImageControl.ImageControl.LoadImageFromFile(String fileName, Boolean readOnly)

  • Some image attachments will not open in EncounterPRO

Cause

This error occurs when GDI+, the imaging library previously used in EPImageViewer, cannot recognize the format of an image.

Workaround


Resolution

The latest EPImageViewer uses a different internal imaging library, which handles many more image and compression formats.

The latest EPImageViewer can be installed by upgrading EproLibNET, which can always be found on ftp://ftp.jmjtech.com in the EP_SupportingInstallers directory.
