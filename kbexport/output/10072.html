Services not starting-- invalid password

10072

Introduction


Symptom

When trying to start EncounterPRO Service(s), get this error:



Trying to open Active Directory and Users to change the password of the service's username there gives this error:


Cause


Workaround

Go to the properties of each service experiencing this problem, click on the tab named Log On, Browse, find the username for this service, re-enter the password, Apply, then the service will start.

--

If that doesn't work reinstall epro.

Resolution


Notes
