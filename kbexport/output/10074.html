Patients not showing from Waiting room

10074

Introduction


Symptom

A patient is found in the Waiting room, but when going to an Exam Room and clicking on Get Patient, patient is not listed.

Cause

This usually happens when the Get Patient service has already been completed.
To confirm:

  1. Get Encounter ID from p_patient_Encounter
  2. Filter for correct patient and encounter_id in p_patient_wp_item
  3. Filter for ordered_service = GET_PATIENT
  4. Status will usually be COMPLETED

Furthermore, this may be caused by a misconfigured PM integration ORby rechecking a patient that has been checked out.
 

Workaround

Advise client to go to the Waiting Room Menu and move the patient tothe Waiting Room by using the "Change Room" icon. If there is no iconon this menu follow the below steps.

  1. Select Waiting Room
  2. Select Patient Name
  3. Select Standard Waiting Room Menu Default or approipate menu
    • (IF you do not see this, go to KB 10328)
  4. Select Edit Menu (IF you do not see this, click on Create Local Copy and edit that Local Version)
  5. Select New
  6. Select Service
  7. Select Encounter
  8. Select Change the Patient Room
  9. Select Finish
  10. Change Button Title to Change Room
  11. Select OK

Resolution

Add a a Change Room button on the waiting room patient menu.
