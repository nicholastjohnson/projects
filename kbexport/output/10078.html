Stopping sp_rebuild_constraints job

10078

Introduction

This document provides instructions on how to stop running sp_rebuild_constraints

Symptom

Practice needs to use EPRO and the job needs to be stopped.

Cause

Workaround

Resolution

Follow the below instructions

  1. Open up Query Anzayler
  2. Run dbcc opentran ('cpr')
  3. It will list what table the database is currently rebuilding
  4. Stop Rebuild job in SQL
  5. Rebuild the table that it was currently rebuilding for example Sp_rebuild_constraints ‘p_observation’
