Setting up Signature Stamp Preferences

10080

Introduction

This article provides information on how to set up the database preferences that will enable signature stamp to be displayed.

Symptom


Cause


Workaround


Resolution

VS 4.0

There are two preferences to set up signature stamp.

To set up up the signature stamp to print always even when there is a manual signature obtained follow the below instructions.

  1. Add below record to o_preferences
  2. Import Signature into User Name in EncounterPRO through Configuration\Users
  3. Restart Report Service
  4. Restart EncounterPRO

VS 2.0

There are two preferences to set up signature stamp.

To set up up the signature stamp to always print

  1. Add below record to o_preferences
  2. Add signauture in bitmap format to CPRFILES\BITMAPS
  3. Add file name to c_provider in signature file colum. Example JMJ.bmp
  4. Restart Report Service
  5. Restart EncounterPRO
