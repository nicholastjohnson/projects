Adding Patient referred to and Specimen Send to to a new template

10081

Introduction

Symptom

How to add the Patient Refer to button and Speciment Send to button to certain labs

Cause

Workaround

Resolution

Before a customer receives their database a script has been run to add the “specimen sent to:” and the “patient referred to” option to all the send-out/refer out treatments.

After you receive your database, if you create any new labs, test, x-ray, etc template in your database, the following steps needs to be performed manually to each template that were created.

                        From the office view screen

                        Select the Configuration tab

                        Select the observation tab

                        Select the respective treatment option to be modified from the list

                        Select the lab or treatment from the shortlist, if the treatment is not listed there use your search options to find the treatment to be modified.

 

Once you select the correct treatment a menu with different options will come up.

                        Select edit observation and you should be at the observation definition screen 

                        Select either the Results or the perform button (depending on which template format you are in, simple vs. composite).

                        Select Property as your option 

                        Select the new result button

                        Select the EncounterPRO Property button and chose user defined as the option

                        Select either “specimen sent to” or “Patient referred to” as your option

                        Select the service box and chose “Treatment Referral” as your service option. 

 

Please refer the same step to add the second property.

Your treatment should now have the “Patient Referred To:” and “ Specimen sent to:” should now show up at the top of the newly created templates on the collect specimen service bar for the nurse to collect the specimen.

 

 

 

 

 

Notes

To add to all treatment types

execute sp_add_property_results_to_treatment_type 'LAB', 'Patient Referred To:'
