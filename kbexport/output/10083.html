Changing the Favorite Chart Display Mode

10083

Introduction

Symptom

The default SOAP screen view is not set up to display Assessments and Treatments paired together, or the SOAP screen defaults to displaying only Assessments.

Cause

Workaround

Resolution

1. Click on the Configuration tab in Encounter Pro.

2. Select the Preferences button.

3. Set the Prefrence Type to User Defined Properties

4. Select the Global button at the bottom left. This will change the setting for all users.

5. Now select 000 Standard Favorite Chart Display Mode.

6. Type "ATAT" in the text box shown

7. Repeat, after highlighting the Specialty button at the bottom

8. Click finished to exit to the Preferences screen

9. Click OK to complete the procedure and return to the configuration screen.

10. You're done!

Notes

If there is no preference for Chart display mode, go to c_chart_section_page_attribute and look for attribute = "display_mode".  Change the values as needed.
