MIK mapping for EPROSERVER

10089

Introduction

This explains how to view and edit the MIK mapping for EPROSERVER.

Symptom


Cause


Workaround


Resolution

If Millbrook / Centricity is installed at a practice, there should be a link on the desktop to a file named ServerSetup.exe.
The file path will be something like this:
E:\Install\Millbrook\CPOPM03\Disk2\SS\ServerSetup.exe

When opening this, select Millbrook Integration Kit Configuration,Next, Next, Next, Next, Next, select EPROSERVER, Modify, Mapping.

Here is where you will find the settings of all mapped compenents. Forany component you wish to analyze further and/or modify, select it thenpress Modify.
