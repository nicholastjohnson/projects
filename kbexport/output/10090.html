Unwanted Folder Selection Steps for Vaccine Signatures

10090

Introduction

In some databases, there have been complaints that when a user provides a vaccine signature, screens come up directing the user to select a folder and to send a TODO item.  This is caused by a record in the c_folder_selection table.

Symptom

When a user provides a vaccine signature, screens come up directing the user to select a folder and to send a TODO item

Cause

The cause is a record in the c_folder_selection table.  The record will look something like the screen below:

 

dbo_c_Folder_Selection
folder_selection_idcontext_objectcontext_object_typeattachment_typeextensionfolderauto_select_flagsort_sequence
1782TreatmentIMMUNIZATION

Other Treatment RecordsY220

 

 

Workaround

Resolution

Remove the IMMUNIZATION record from the c_folder_selection table.

Notes

Problem may be present in Production Configuration Version 4.05.19
