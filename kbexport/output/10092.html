Create restart services job

10092

Introduction

This article details how to create a job in SQL Server Enterprise Manager for automatically restarting the EncounterPRO service(s)

Symptom

Practice complains about services needing to be restarted (or some other complaint, the solution to which involves restarting services) on a regular basis.

Cause


Workaround

Restart services manually.

Resolution

Download restart job from \\Wwwserver\Ftproot\EP_Misc_instructions\Programs\RestartJobs and place in \\server\CPRFILES

Follow the below steps

  • Open up Enterprise Manager
  • Select Tools
  • Select Job Scheduling
  • Select Next
  • Select Operating-System shell command ...
  • Select Next enter the path to the file.
        (Example: D:\JMJ\recycle_encounter.bat)
  • Select "Next" 
  • Select "On Recurring Basis"
  • Select Schedule 
  • Set Job to run Daily at 7 am. If practice reboots after 7 am set time to start right have server is rebooted
  • Select Next and Next again
  • Rename Job "EPRO Restart Jobs"
  • Select Finish and OK

Also run correct Dependencies.reg file
