Error clicking Utility or Configuration item

10094

Introduction

 

Symptom

When the user clicks on a configuration option or a utility option, nothing happens and an error such as the following is generated:

Event Type:       Error
Event Source:    EncounterPRO
Event Category: None
Event ID:           0
Date:                12/17/2004
Time:                11:15:52 AM
User:                N/A
Computer:         CHRIS-VM-XPPRO
Description:
CHRIS-VM-XPPRO\Administrator on CHRIS-VM-XPPRO
4.05.33.7 (10/8/2004) >>> UNKNOWN CALLER - (f_do_menu_item()) Menu Item not found (8, 1000160)

Cause

The Utilities and Configuration options tabs are driven by menus ("Standard Utility Menu Default" and "Standard Config Menu Default" respectively).  In EncounterPRO 33.7 and earlier builds, these tabs are unable to handle local copies of either of these menus.

Workaround

Execute the following query.  This will disable the local copies of the utility and configuration menus.

UPDATE c_Menu
SET status = CASE WHEN menu_id >= 1000000 THEN 'NA' ELSE 'OK' END
WHERE description IN ('Standard Config Menu Default', 'Standard Utility Menu Default')
GO

Resolution

Upgrade to client 33.15 or later.
