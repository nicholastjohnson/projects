Particular Drug does not automatically print

10099

Introduction

This article provides instructions on what do when a particular drug does not automatically print.

Symptom

Cause

This happens when the default treatment mode is set to "OTC no Print" on a particular drug

Workaround

Resolution

To set a drug to automatically print the following needs to be done

  1. Order drug that is not printing
  2. The drug will show under selected items and will probably say "OTC no Print"
  3. Select Drug from Selected Items and select "Default Treatment Mode"
  4. Select OK
  5. Confirm that Drug printed automatically

Notes
