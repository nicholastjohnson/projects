User cannot delete PFSH items

10100

Introduction

The user attempts to delete one or more PFSH items, but they never disappear from the PFSH review screen.

Symptom

If the user reviews a PFSH and wishes to remove an item, they click on the results pane and select "Reviewed with changes" and on the edit screen they drill down as appropriate and remove an item.  The item appears to be removed but when they exit the edit screen the PFSH review screen still shows the removed items.

Cause

This problem is caused by the logic of the PFSH display, which attempts to locate and display the last time each PFSH "question" was answered.  When an item is deleted, the process does not see the deleted item so it goes back in time to when the item was not deleted and shows that answer.

Workaround

Apply Hotfix 33.

Resolution

Either upgrade the database to mod level 124 or apply Hotfix 33.
