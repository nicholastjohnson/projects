Restoring assessment risk levels

10105

Introduction

Sometimes after upgrades, the assessment risk levels need to be put back.

Symptom

Cause

Workaround

Resolution

Run this script to restore script levels - be sure to make sure the old db name is correct.

UPDATE c

SET risk_level = x.risk_level

FROM c_assessment_definition c

            INNER JOIN cpr_pre_337.dbo.c_assessment_definition x

            ON c.assessment_id = x.assessment_id

WHERE c.risk_level <> x.risk_level

OR (c.risk_level IS NULL AND x.risk_level IS NOT NULL)

OR (x.risk_level IS NULL AND c.risk_level IS NOT NULL)
