Patient Chart does not display information on SOAP screen

10115

Introduction

This article provides information when you click on Patient Record and there is nothing in the big grey box

Symptom

See attached screens

 

Cause

This is caused ultimately by really complex signatures which cause the CIC software to create a biometric token greater than 4099 characters.  The 4099 limitation is caused by a datawindow field size in PowerBuilder. We have increased it to 32000, but that wonâ€™t take effect until build 4.0

Workaround

To see which signatures are too large follow the below steps

  • Run below script in Query Anazlyer

Select cpr_id, attachment_id

from p_attachment_object

where datalenght (object_test) >4099

  • You will see two columns. The first column is the CPR_ID and the second is the attachment ID
  • Filter for CPR_ID in p_attachment object.
  • Highlight record for the attachment_ID and see who created this record in the created_by field
  • You will need to find another signature from the same User_ID in the created_by field
  • Copy good signature in the offending field
  • Open chart and confirm that it looks correct
