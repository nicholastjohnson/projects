Adding Users

10119

Introduction

This article explains how to add licensed users to an EncounterPROinstallation running on a version 4 database. That is, this article isonly for adding Physicians (doctors) or Extenders (PA, NP, and othermid level licensed users). Any other type of user should be added bythe practice; the practice should know (or soon be taught) how to dothis via Trainers.

Symptom


Cause


Workaround


Resolution

Go into the EncounterPRO application/interface, Configuration, Users,New User, enter the user's full name, short name, & initials.Specify the user's color, Primary Office, specialty (usually the same as the practice'sspecialty), and Provider Class.

Click Save

Specify the user's roles. Under the Provider Information section, enterthe user's first name, middle initial, last name, degree, DEA# andLicense as they are available. Set Rx Certified to Yes.

Click Access Code and enter the user's access code twice. If the userdid not provide an access code, and this is a new practice, make up anaccess code and specify it on the sheet detailing which users to add.

Click Save

Click Privildges and move "User has all priviledge..." to the right.

If this is the first user being created for this practice, click on theSupervisor button and for each user with a blank name, click on thegrey square to the left of them and select Remove Short List. Thenclick Cancel.

Click Save

Notes

If the user has a signature to be scanned in, see articles 10080 & 10103 for further info.
