Creating a user license

10122

Introduction

This article explains how to add a user license in version 2.

Symptom

Practice needs to add, subtract or modify the license of a provider (physician/doctor) or an extender (NP or PA, etc).

Cause


Workaround


Resolution

Go to here:
\\Techserv\production\License Manager
and run this:
eclicense.exe

In here, find the practice, edit their license accordingly. Make surethe info in this practice's profile matches up with our profile forthem in Outlook and the info in their database. The database should bethe deciding factor on what the values are in all other places.

After you edit the profile for this practice in eclicense.exe, updatethe profile. This will create a file on a floppy disk named after thepractice. Change the name of this file to jmjlic.ini and place thisfile in the cprfile folder at teh practice (via email or ftp). Restartthe license service at the practice.
