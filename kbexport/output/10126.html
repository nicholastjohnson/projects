Pre Version 2 Upgrade Check List

1.                 Check disk space availability – Do not need a lot ofspace to do the upgrade but if they are low in space they need to be informed.

2.                 Check scanned documents for size of files.  If large teach them how to scan properly and recommendhaving their image files shrunk.  Thereis a fee for this.

3.                 Download or Mail CD

a.       Ifmailing they  need to get the CD inenough time to be able to go into the site at least one day before the upgradeand copy the contents of the CD to the hard drive.

4.                 Check server for the following

a.       UpdatedVirus software

b.      Latestwindows updates for both the servers and the workstations

c.       Latestservice packs for SQL 7 or SQL 2000 whichever they are on (Can be found on theWindows Updates scan). (If SQL 7, the MS SQL Server will say v7)

                                                              i.     If they are on SQL 7 recommend that they upgrade to SQL2000 ASAP.  This will be mandatory beforethe Version 2 to Version 4 conversion.

5.                 Get and emergency contact number for someone in theoffice that can get back into the office after hours in case we have a problemwith the server.  If during businesshours a back line number and confirm someone will be available to answer thephone during scheduled upgrade time.

6.                 Email or Fax release notes

7.                 Verify the server and All fat clients that will berunning EncounterPRO are on Win2K or XP, or at least NT SP4.

Upgrade Procedures 2.21.44

 

Create an issue in Outlook and copy this document into the issue.  While doing the upgrade if there are anyunusually behaviors or problems with the upgrade document within the issuesexactly what happen.

 

 

       1.  Backup of EncounterPRO Database

       2. Stop EncounterPRO Services; Verify you know the SA password 

       3. Copy entire contents of EncounterPRO CD to the Hard Drive.

a.       Donot remove the old EncounterPRO CD until the upgrade is complete and working.

b.      Ifthere is not already, create a directory on the data drive called Install and thencreate a folder under Install called JMJ then place the copy of the CD in theJMJ directory using this format, CD2.21.44

 

       4. Install EncounterPRO Client

a.       Ifterminal server you will need to put the session in Install mode

                                                                    i.     Go to start run and type:  change user /install

 

       5.  Restart Server if prompted

a.       Whenthe server comes back up you will get a message saying that a service failed tostart.  It is the EncounterPRO servicethat is not starting because the database has not been upgraded yet.  Just ok the message and continue with theupgrade process.

b.      Ifterminal server and there is no reboot then put system back into execution modebefore proceeding.

                                                                    i.           Go to start run and type:  change user /execute

       6. Update Database

a.       Runthe server manager (\install\jmj\cd2.21.43\EncounterPRO\cprserver.exe) and tellit yes when it asks if you want to upgrade.

b.      Oncethe upgrade scripts have run successfully the server manager should give you amessage of success.

c.       Youwill get a message “Copy Scripts Failed” so you will need to copy the clientinstall files to the install directory under CPRFILES

a.       Youwill need to create a folder under the install directory called 38 and copy theclient files into this directory

b.      Thencopy over the MSVBVM60.dll and Setup.exe files to the install directory

d.      Closethe Server Manager, disregard any missing License Info

 

       7.  Verify Database Backup Configuration to NewFormat

Delete the SQL Joband Maint Plan for cpr. Run “Create DB Maintenance Scripts Version 2” inQuery Analyzer against cpr

 

       8. Check integration to make sure it is still working by

a.   No Integration
