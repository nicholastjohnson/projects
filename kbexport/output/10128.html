Default encounter type for PM integration

10128

Introduction

 This article describes how to set the default encounter type for practices using a Practice Management integration software.

Symptom

Cause

Workaround

Resolution

 The default encounter type for non-hl7 interfaces [like medisys,medical manager,fox meadows,lytec,medisoft] is set as 'SICK'. If you need to change this to some thing else then add or edit the following row in c_component_attribute table.

component_idattribute_sequenceattributevalue
MSGHANDLERSERVER1000079default_encounter_typeenter name of encounter type

 

For HL7 interface, it's set up in o_Preference Table as preference_id of 'Default_Encounter_type'.
