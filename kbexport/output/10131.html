No active location domain error

10131

Introduction

This article describes what to do if an assessment gives an error about no active location domain

Symptom

Clicking on a service bar for an assessment gives an error like this:


Cause


All locations for this assessment are non-existent or deactivated.

Workaround


Resolution


Go to c_assessment_definition, find the assessment in question and find its location_domain. Find the row(s) for this location_domain in c_location_domain and in c_location.

If there exists a row for this location but it is set to NA, set it to OK.

Else, if there exists a row for this location that is not set to NA, verify the problem still exists.

Else, contact JMJ support for further instructions.

Notes


see also Support issue 17213
