Setting up job to delete jmj temp files

10133

Introduction

This article provides information on how to set up a job on the server or fat client to delete jmj temp files on a weekly basis.

Symptom

Below are some symptoms

  • User cannot scan or post
  • User cannot view images
  • User locks up when scanning or posting

Cause

The usually cause is that the temp files has too many files

Workaround

Resolution

  • Download and unzip purge.zip from FTP or EPRO CD \EP_Misc_instructions\Programs\Purge
  • Create folder C:\Purge
  • Copy unzipped files to C:\Purge

For a Fat Client

  1. Select Start\Programs\Accessories\System Tools\Scheduled Tasks
  2. Select Add Scheduled Task
  3. Select Next
  4. Select Browse and browse to C:\Purge\clearjmjtemp.bat
  5. Select When my computer starts
  6. Enter in username and pw
  7. Select Next and Finish
  8. Right click on job and Select Run
  9. Ensure that job runs

For the server

  1. Open up Enterprise Manager
  2. Select Tools
  3. Select Job Scheduling
  4. Select Next
  5. Select Operating-system shell command ...
  6. Select Next
  7. Type in box "c:\purge\clearjmjtemp.bat"
  8. Select Next
  9. Select on a Recurring basis
  10. Select Weekly on Sunday at 10 PM
  11. Select Next and Next again
  12. Rename Job "EPRO Clear Temp"
  13. Select Finish and OK
  14. Right click on Job and Select Start Job
  15. Confirm that it works
