Procedure did not bill

10135

Introduction

This article explains the different possible reasons why the procedure is not billed

Symptom

Procedure is not showing in the billing statement or review billing in encounterpro.

Cause

 

Workaround

  • It's possible that the procedure is set to "do not bill". Look in configuration for procedures and see whether they are set tobill.
  • For "Medicaid" patients when they are combining "WELL/SICK"diagnosis, by default EncounterPro allows only first SICK diagnosis andtreatments attached to it and suppresses the other sick diagnosis createdon that encounter. If the state Medicaid accepts more than one sickdiagnosis then get help from Sumathi or David on how to allow everythingto be billed.
