Creating Rebuild Constraints Job

10136

Introduction

This article provides information on how to create the rebuild constraints job

Symptom

Cause

Workaround

Resolution

Follow the below steps

  • Open up Enterprise Manager
  • Select Tools
  • Select Job Scheduling
  • Select Next
  • Select Transact-SQL command
  • Select Next
  • Select Database name usually cpr
  • Type in box sp_rebuild_constraints '%'
  • Select Next
  • Select on One time and enter in Date 
  • Select Next and Next again
  • Rename Job "EPRO Rebuild Constraints"
  • Select Finish and OK
  • Dial in the following morning after job has been run and confirm that it runs
