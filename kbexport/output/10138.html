Well visits getting encounter code for wrong age group

10138

Introduction

Symptom

Customers complain of encounters being coded as Well 12-17y or 5-11y when are all adults.

Cause

There are no well diagnoses made for the visit.

Workaround

Resolution

We most often see this for OB/GYN visits:  if there is no well diagnosis, then the program code is not triggered that sets the encounter code according to age range.
