Introduction

This isn't just for followups - you can replace FOLLOWUP with any treatment type you like.

Remember, these Outstanding screens in the Utility menu only show open treatments, so please change the default treatment workplan and/or Workplan Close toggle on the Treatment Type definition screen if necessary.

Adding "Followup"s to Outstanding Treatments in the Utilities Menu

Go to o_menu_selection and search for menu_context = Utility

The Menu_id in that record is the menu_id for the Utilities screen.

Go to c_menu_item and search for that id.  Get the menu_item_id for Outstanding Other Treatments.

Search for that menu_item_id in c_menu_item_attribute.

Look for the record with the attribute of observation_type.  The value of that record is the observation_type by which that screen is filtered.

Go to c_treatment_type and filter for treatment_type "Followup".

Insert the value you found in the last table into the observation_type field.

 

That's it!
