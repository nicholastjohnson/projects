General Commands

 

Command

Argument

Description

Note

format

Applies formatting command to change text font, style, size, color, justification, etc.

 

print

Text to be printed

prints argument as literal text

 

blank line

#

Adds enough carriage returns to make # blank lines after the previous non-empty line.  # defaults to 1.

 

cr if line not empty

 

Adds carriage return if current line if not empty

 

cr

#

Adds # carriage returns.  Adds 1 carriage return if # is not specified

 

tab

 

Adds tab character

 

space

#

Adds # spaces.  Adds 1 space if # if not specified.

 

remove line

#

Removes # lines.  Removes 1 line if # is not provided.

 

remove character

#

Removes # characters.  Removes 1 character if # is not provided.

 

property

Adds text generated by EncounterPRO Property

 

page

 

Add page break

 

header

 

Sets insertion point to document header

 

footer

 

Sets insertion point to document footer

 

body

 

Sets insertion point to document body

 

date

 

Adds current date

 

time

 

Adds current time
