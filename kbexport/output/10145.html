Creating a new Outstanding Treatment item in Utilities

10145

Introduction

I'm writing this using the example of FOLLOWUP, but of course you can substitute other treatment types.

Symptom

Cause

Workaround

Resolution

Notes

Go to o_menu_selection and filter for context = Utilities.  The menu_id is the menu_id for the Utilities screen.

Go to c_menu_item and filter for the menu_id above.

Copy the record for Outstanding Other Treatments and change the button_title to Outstanding Followups.  Note the menu_item_id and adjust the sort sequence.

Create new record in c_menu_item_attribute  - enter the menu_id and menu_item_id from above.  Make the attribute = observation_type and the value = Followup.  Create another recors with the same menu_id and menu_item_id with attribute = get_labresults and value = TREATMENT_REVIEW.

Go to c_treatment_type and filter for treatment_type Followup.  Enter Followup in the observation_type field.

Go to c_observation_type and make a new record for Followup following the same guide as Referral.

 

That should be it.
