Setting up Spirometer integration

10146

Introduction

This article details how to set up spirometer integration.
See also KB 10196

Symptom


Cause


Workaround


Resolution

For each computer that must generate or view Brentwood Spirometry reports:

 

  1. Make sure that any existing Midmark or IQMarksoftware is removed from the computer, including but not limited to anyMidmark dependencies installed with pre-4.05.34 versions ofEncounterPRO.
  2. Installthe Midmark 6.3 SDK. (\\Wwwserver\ftproot\EP_Upgrades\Production\Encounter_Production_CD\JMJ_Support\Programs\IQMarkSDK633)
  3. You will find six (6) .RAR files. Extract any one of these and run the setup.exe file
  4. Install the latest EproLib4 and EproLibNET libraries.
  5. Ensure the following exists in c_Component_Registry:
