How to restore a cancelled encounter uncancel

Workaround


Resolution

If the customer is on Mod Level 126 or later, execute the following Stored Procedure in a query window:

 

EXECUTE jmj_uncancel_encounter 'xxxxx' , yyyyy


... where xxxxx is the CPR_ID and yyyy is the ENCOUNTER_ID for the cancelled encounter.


Note, cpr_id is a string and thus must be enclosed with single quotes(' ').  Encounter_ID is an integer and thus should not be enclosed byquotes.

Also, make sure you have run hotfix 97 against the database.
 

If the customer is on Mod Level 125 or earlier, perform the following:


Upgrade the database to Mod Level 126
Run DBSCRIPTS
Run all hotfixes

Follow instructions above
