Staying Logged on in EncounterPro

10149

Introduction

Symptom

Cause

Workaround

Resolution

Setting up a computer to use Computer Secure (allows user to stay logged on longer)

 

  • Open up EncounterPRO with a user that has all privileges
  • Select “Configuration”
  • Select “Preferences”
  • Change Level from “User” to “System”
  • Ensure that Preference Type = General Preferences
  • Select “New Preference”
  • Select “Computer Secure”
  • Enter in “True”
  • Select “OK”
  • Select “Exit”
  • Select Logoff
  • Close and reopen EncounterPRO
