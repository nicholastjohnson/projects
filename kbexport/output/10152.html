Setting up Patient Education materials to print automatically to the local default printer

10152

Introduction

Symptom

If a practice wants their Patient education materials to print automatically

 the following needs to be set up.

 

Cause

Workaround

Resolution

This fix has already been applied to the master as an option when configuring that service.

But for current sites and new client the following steps nees to be done from the tables.

1. look for the o_service table, and search for "Patient Materials" under service description.

2. Copy the service name for that discription (ex: PATIENTMTRL)

3. go to the o_service_attribute table and create a new field for the service name from step # 2 (ex:PATIENTMTRL)

it should look like this. this will cause the pt edication material to automatically print.

dbo_o_Service_Attribute
serviceattribute_sequenceuser_idattributevalue
PATIENTMTRLautonumber
actionPrint

To print the material automatically for a specific user then Fill in "user_id" field on the o_service_attribute with that specific user id. After configuring this test to make sure it works.

dbo_o_Service_Attribute
serviceattribute_sequenceuser_idattributevalue
PATIENTMTRLautonumber001Test
actionPrint
