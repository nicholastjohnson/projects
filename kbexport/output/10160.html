New Allergy Button does not work

10160

Introduction

This article provides information on when the New Allergy button does not work

Symptom

Cause

This usually only happens after an upgrade takes place and it is because the menu context is set incorrectly.

Workaround

Resolution

This can be resolved by changing the menu context on the menu bar or by running the below script

UPDATE c_Menu

SET context_object = 'Patient'

WHERE description LIKE 'Standard ALLERGY (Assessment List Service)%'

AND context_object <> 'Patient'

GO
