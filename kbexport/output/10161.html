Unable to view all or certain entry on Devel Tab in 2.0

10161

Introduction


Symptom

  • Certain values do not show on Growth Tab. For Example Percentage, HGT, WGT, HC
  • Clicking on the Devel Tab gives the below error msg

The description for Event ID ( 0 ) in Source ( EncounterPRO ) cannot be found. The local computer may not have the necessary registry information or message DLL files to display messages from a remote computer. The following information is part of the event: Unknown Caller - (SYSTEM ERROR) errornum:1

message :Divide by zero at line 18 in function f_get_bmi of object f_get_bmi.

where   :f_get_bmi

object  :f_get_bmi

event   :f_get_bmi

line    :18.

Cause

There is usually a vital entry of 0

Workaround


Resolution

Search for the patient's billing ID in p_objective_result and remove any records where the result value is 0. This problem is fixed in 4.0
