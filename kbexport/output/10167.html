Icons Missing From EncounterPRO 33.7

10167

Introduction

EncounterPRO 33.7 might have some missing icons with Config Verions 19 or later.

Symptom

For some buttons, the picture is blank.

Cause

Config Version 19 and later use some new icons which were not compiled into EncounterPRO 33.7.

Workaround

Navigate to the following FTP folder:

ftp://ftp.jmjtech.com/EP_Quick_Fixes

Download the EproIcons.zip file and unzip it into the program directory (where EncounterPRO.exe resides).
