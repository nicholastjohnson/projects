Troubleshooting billing not crossing for one patient

10171

Introduction

This article describes how to triage an issue in which the billing for one or more (but not all) patients is not crossing.

 

NOTE: This is a work in progress.

Symptom

  1. An encounter's billing does cross, but some billable items don't show

Cause

  1. Encounter was closed and reopened, then a billable item was added, then the encounter was closed again.

Workaround

For each patient example, if the patient has been checked out, find out what the icon is next to the patient.

If the icon is a green check mark, billing posted, which means thebill crossed over. In this case, go to jmj\pmshare\Hold Bills and findthe file for this patient's encounter. Paste this file into the issue.

  1. Retry Posting in order for the changes to go over
