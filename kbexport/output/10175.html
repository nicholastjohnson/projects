Immunization report printing same patient data for all patients.

10175

Introduction

This article will provide instructions on replacing a report in a Version 4 DB.  Such reports include the GA Immunization report, Supplemental Immune Report, etc.

Symptom

Cause

Practice clicked Yes when prompted to save changes after printing the report for that one patient.

Workaround

There are two ways you can install the Reports Installer program on to the server.

  • You can go to the JMJ FTP site at ftp.jmjtech.com and follow the below instructions:
    1. Open the EP_Reports folder
    2. Open the 4.0 floder
    3. Copy the Extract ReportInstaller file to a location on the server.
  • You can also install the Report Installer from the EncounterPRO installation
    1. Go to the CPRFiles location on the server which usually can be found at \\(Server name)\Install\setup.exe
    2. Click Next
    3. Select Modify, Then Next
    4. Click Next
    5. Click Next
    6. Make sure there is a hardrive pic next to the ReportInstaller option, Then Click Next
    7. Click Install

Once you have installed the report Installer please run the Installer either from the location you installed it in Option 1 or the if you selected Option 2 to install it is located in the same directory you installed the EncounterPRO application (ex. C:\Program Files\JMJ\EncounterPRO\Report Installer).

The Following Instructions are for installing the report:

    1. Double click on either the Extract ReportInstaller.exe file or the ReportInstaller.exe file depending on the option installed
    2. Type in the Server name
    3. Type in the Database name
    4. Click OK
    5. Select the Report(s) that you wish to install
    6. DO NOT select the box that erases current report settings for this will erase all of your current report settings.
    7. Click Finish
    8. You should get a message that tells you that you have successfully installed the reports
    9. After installing you may have to go in an reconfigure your setup of the reports

 

Resolution

Instruct the practice to always click "No" when they close the report and are prompted for whether to save changes to the report after previewing it.
