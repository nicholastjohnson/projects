Installing multiple versions of 4.0 on one workstation

10176

Introduction

This article provides information on how to have mutliple versions of 4.0 on one PC

 

Symptom

Cause

Workaround

Resolution

Note: Steps 1, 12 and 16 are very important.

 

  1. Change Name of EncounterPRO on desktop. For example “EPRO 33.7”
  2. Install other Version 4.0 from \\Techserv\production or \\Techserv\Testing
  3. Run Setup.exe
  4. Click “Next”
  5. Accept Agreement
  6. Click “Next”
  7. Enter in Database Server name
  8. Database = cpr_40
  9. Click “Next”
  10. User name and Organization can be anything
  11. Install Application for “Anyone who uses this computer (All Users)”
  12. Install Program to new folder for example “C:\program files\jmj\Encounterpro_33.18”
  13. Click “Next”
  14. Click “Next”
  15. Select “Finish”
  16. Change icon on Desktop from “EncounterPRO” for example “EPRO 33.18”
  17. Confirm that EPRO 33.18 works by double clicking on icon
  18. Confirm that EPRO 33.7 works by double clicking on icon
