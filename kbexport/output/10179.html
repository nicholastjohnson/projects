Unable to start transport out service - ERROR Type mimatch accessing external object property

10179

Introduction

This article explains why the type mismatch error is thrown while starting transport out service.

Symptom

4.05.33.7 (10/8/2004) >>> u_component_serverservice_outgoing - (timer) Error in timer_ding function (Type mismatch accessing external object property value at line 291 in function chrg of object u_component_outgoing_filecopy_hl7.)

Cause

Mostly this error shows up if the following attribute is left NULL.

c_Component_Attribute
component_idattribute_sequenceattributevalue
TRN_OUT_FILECOPY_HL71000030FACILITYID

Workaround

Fill in a valid facility id value to fix this error and restart the transport out.

Resolution


Notes

the above mentioned issue is no longer valid from Install Bundle 5.1.5 because it does not care about "FacilityID" attribute for "TRN_OUT_FILECOPY_HL7".
