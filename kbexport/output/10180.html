Billing Courtesy options on Billing Edit screen

10180

Introduction

The options on the Billing Courtesy popup list are editable.

Symptom

Cause

Workaround

To edit the options on the Billing Courtesy popup list, filter the c_Domain table on domain_id = 'COURTESY_CODE' and add/remove/edit records as needed.  The [domain_item] column holds the code that will actually be saved in the p_Patient_Encounter tables and sent to billing systems.  The [domain_item_description] column holds the value that is actually displayed to the user.
