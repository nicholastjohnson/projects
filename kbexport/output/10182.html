Upgrading from one Build to another

 

1)      Right Click on the CDDROM drive and select Explore

2)      Go to the following folder Y:\Cd4.05.32\EncounterPRO\client\33.7(this may be different depending on your version of EncounterPRO)

3)      Double click on Setup.exe

4)      Select “NEXT”

5)      Select “I accept the terms in the license Agreement”

6)      Select “NEXT”

7)      Enter in Server Name and Database Name (if none are known enter nothing)

8)      Select “NEXT”

9)      Select “NEXT”

10)  Select “NEXT”

11)   Select Typical

      14)  Select “NEXT”

15)  Select “INSTALL

16)  Select “Finished”

17)  If prompted for Reboot. Reboot the computer
