The Importance of Keeping Good Backups

  • Tape Backups Are The Last Line Of Defense Against Data Loss
  • You Can Restore Your Entire Data Infrastructure From A Single Tape
  • If You Do Not Maintain A Good Backup, A Server Crash Can Be Catastrophic

 

 

JMJ Tape Rotation Recommendation

  • Every day that EncounterPRO is used, make a backup that night
  • Take all tapes off site within 24 hours.
  • Every 7 days and on the same day every week, take a tape and store off-site for 4 weeks.  The other tapes can be reused.
  • Once every month, remove a tape from rotation and store off-site for the duration of your obligation to keep medical records.  This varies by state and specialty.
    • Suggest using the tape from the 1st of every month.
  • If you run a 5 day a week practice this requires
    • 12 monthly tapes per year never reused.
    • 4 weekly tapes that are recycled

  • Replace tapes annually or after 50 use cycles, which ever comes first.
  • Highly recommend using professional off-site storage service such as Iron Mountain for monthly tapes and possible weekly tapes.

 

How To Make Sure Your Backup Is Working

  • Set A Non-critical File To Back Up During The Night
  • Restore That File From A Random Media Set
  • Check File Integrity
