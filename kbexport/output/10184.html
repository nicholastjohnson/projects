Installation

  1. Close all open programs.
  2. Reboot computer.
  3. Log on computer as administrator.
    1. Click “Start”, then “Settings.”, then “Control Panel”, then “Add and Remove Programs”  
  4. Find EncounterPRO and select “Change”
  5. A window will open starting the upgrade process. Click “Next.”
  6. Select “Modify”
  7. Select “Next”
  8. Confirm the server name and the database name (typically CPR).
  9. Click “Next” again.  
    1. Click the Red “X” beside MS Calendar Control and select “This feature, and all sub-features, will be installed on the local hard drive.”
  10. Click “Next..
  11. Click “Install.”  The program will install; when it is complete, click “Finish.”
  12. Reboot the computer.
  13. Log in normally and test EncounterPRO.

 

If any problems result from this fix or persist after reinstallation, please call JMJ software support for assistance.
