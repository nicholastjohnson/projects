1. Install Client (follow instructions in Installing EncounterPRO)

2  Restore Database
a.         Restore DB to the drive and folder created for the live data

3. Drop & Create jmjtech user
a.         Run script called set securities. txt

4. Create CPRILES Directory
a.         Create a folder on the D Drive call JMJ

5. Create Services
a.         First you need to create the users that will run the services (password for all is !Epro&& and they must be a member of the domain administration.)  

a. For 4.05 sites the user name should  be eprojmj

b. For 4.04 sites the user name should be eprolic0001, epro0002, epro0003, epro0004, eprobill0001, eprosched0001, eprolic0001.

c. You can also use the bat files on the FTP site which can be found under ftp://www.jmjtech.com/Support/Services/.  Depending on what intregration and version the practice is running depeneds on what bat file should be run. b.         To create the services manually go to the c:\ dos prompt and type the following:

d. Sometimes the bat files will not work and you will have to manually run the command (Enter between each command line and you should get a success message each time.  (Replace ( ) with appropriate information))

INSTSRV EncounterPRO_Report_1 “d:\program files\jmj\encounterpro\encounterpro.exe” –a (domain name)\epro0001 –p epro

c.         Once the services are created you will have to go directly to each service and manually re-enter the passwords.  Unfortunately Windows will not accept the password that you entered from the DOS prompt.


e.         Start all EncounterPRO Services. Open the application log in the event viewer to see if you get any error messages.  If you do have a problem contact JMJ Support so that they can assist you.

 

            f. Ensure that the EncounterPRO client can successfully open. If you do have a problem contact JMJ Support so that they can assist you.
