Installation

  1. Close all open programs.
  2. Reboot computer.
  3. Log on computer normally.
  4. Click “Start”, then “Run.”  In the space provided type the following:
    1. \\<server name>\cprfiles\install\encounterpro\setup.exe
  5. Click “Ok.”
  6. A window will open starting the upgrade process. Click “Next.”
  7. Accept the terms and click “Next.”
  8. Confirm the server name and the database name (typically CPR).
  9. Confirm the user name (your practice name) then select, ”Anyone who uses this computer.” Click “Next.”
  10. Click “Next” again.  Select “Typical”
  11. Click “Next..
  12. Click “Install.”  The program will install; when it is complete, click “Finish.”
  13. Reboot the computer.
  14. Log in normally and test EncounterPRO.
