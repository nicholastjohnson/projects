Installing EncounterPRO Demo Database

10189

Introduction

This is instructions on how to create a New Demo DB.  Demo DB areused for sales and testing environments.  These types of DB arealready filled with patient data and you are able to select from all ofthe specialties that we offer.  Never use this configuration for aLive Database.

Symptom


Cause


Workaround

1.                  Copy EncounterPRO CD to the hard drive

a.       Create a folder on the data drive called Install

b.      Create a folder under Install called JMJ

c.       Create a folder under JMJ called CD4.05.33

                                                              i.      Copy entire CD into this directory

 

2.                  Install Client

a.       Client setup files located on CD under EncounterPRO\Client\33.7

b.      Run Setup

                                                              i.      If MDAC 2.8 is not installed the wizard will automatically install this version and then require a reboot after it is installed.  Once your reboot the rest of the install will start automatically after you log in.

                                                            ii.      If .NET Framework 1.1 is not installed the wizard will ask you if you want to install.  You must have version 1.1 installed for EncounterPRO to work

                                                           iii.      If the above two requirements are already met you will go directly into the installation wizard for EncounterPRO.

1.      Next

2.      I Accept, Next

3.      Enter Server Name and Database Name (Which should be cpr), Next

4.      Enter any User Name and Organization and leave default Anyone who uses this computer (all users), Next

5.      Install on Data Drive (Usually the D Drive).  Use the default location and just change the drive if necessary, Next

6.      If you need to install Brentwood Runtime then change to Custom otherwise leave as default and select Next

7.      Change the Brentwood Runtime to install and select Next

8.      Install

c.       After the installation is complete the wizard might require a reboot

 

3.                  Restore Database

a.       Create a Database called cpr_4 in SQL Enterprise Manger

                                                              i.      Onthe Data Files tab you will have a file name of cpr_4_Data and you willneed to create another file name of cpr_4_Attachments.

                                                            ii.      On the Transaction Log tab you will have a file name of cpr_4_log.

b.      Copy backup file from CD to Server, if not already done.

                                                              i.      Backup file is locate on the CD under New DB Install\ DB Backup\Config19.bak

                                                            ii.      Once copied to the hard drive unzip

c.       Bringup the Restore properties window and select From Device and go to thepath where you restored the above file and select it.

d.      Now go to the options tab on the Restore properties window.

                                                              i.      Select Force restore over existing database

                                                            ii.      Change physical file name to the following

1.      (DATA DRIVE):\program files\Microsoft SQL server\mssql\data\cpr_4.mdf

2.      (DATA DRIVE):\program files\Microsoft SQL server\mssql\data\cpr_4_attachments.ndf

                                                           iii.      (DATA DRIVE):\program files\Microsoft SQL server\mssql\data\cpr_4_log.ldf

                                                          iv.      Now click OK and the database should start restoring.

 

4.                  Create CPRFiles directory

a.       Go to the New DB Install directory on the CD

b.      If you want the directory on the d drive then execute the file make_cpr_files_dirs.bat

c.       Ifyou want the directory on the c drive then execute the filemake_cpr_files_dirs on C Drive.bat located under the Installation on CDrive folder.

d.      If you want to use any other drive then edit and do the following to the file make_cpr_files_dirs on D Drive.bat

                                                              i.      Do a find/replace for “d:” with the appropriate drive name

                                                            ii.      Save file

e.       Execute the make_cpr_files_dirs on D Drive.bat file

f.        Go to CPRFiles directory location and share folder keeping the folder name as the share name.

 

5.                  Return to SQL Query Analyzer and then locate and run the following script against the cpr_4 database.

a.       The script is located on the CD under New DB Install\4.0 Demo Licensed DB Scripts.sql

                                                              i.      This script will update the 4.0 Master Database to run as a licensed demo

 

6.                  Set Security and Set App Mode in application

a.       Go to the Start Button on the Desktop and select programs then EncounterPRO then EncounterPRO ASK

b.       Click on DB Maintenance and you will get the following screen:

c.      Select the Set Login button and you will get a message “Set Login Succeeded”

d.       Select Set Security App Role ( This will take a minute or two to run)  you should then get a message “Set Security Succeeded”

f.        Close window and edit the EncounterPRO Icon again and change the target back to CLIENT

 

7.                  If you are doing a licensed version you will need to create EncounterPRO Services

a.       Open the document Instructions – Creating Services 4.05.doc
See also KB article 10271
