Creating a new Live DB

10194

Introduction

These are instructions on how to create a new Live 4.05 DB.

Symptom

Cause

Workaround

Resolution

  1. Copy EncounterPRO CD to the hard drive
    1. Create a folder on the data drive called Install
    2. Create a folder under Install called JMJ
    3. Create a folder under JMJ called CD4.05.x.x
      1. Cody entire CD into this directory
  2. Restore DB
    1. Create a Database called cpr in SQL Enterprise Manager
      1. On the Data Files tab you will have a file name of cpr_Data and you will need to create another file name called cpr_Attachments.  You can put both of these on the same drive or separate them out to different drives.  Because the Attachments can grow rather large at some practices it is a good practice to have a drive just for this file group
      2. On the Transaction Log tab you will have a file name of cpr_Log â€“ again you can keep this on the same drive as the data files or put them on a different drive which is recommended
    2. Copy database backup file from CD to Server, if not already done
      1. Backup file is locate on the CD under New DB Install\DBBackup\405_cpr_010804_Master.exe
      2. Once copied to the hard drive unzip(This step may not be necessary)
    3. Bring up the Restore properties window and select from device and go to the path where you restored the above file and select it
    4. Now go to the options tab on the Restore properties window
      1.  Select force restore over existing database

      2.  Change physical file name to the following (Make changes where necessary for site that have the files groups on different drives)

        1. (DATA DRIVE):\program files\Microsoft SQL server\mssql\data\cpr.mdf

        2.  (DATA DRIVE):\program files\Microsoft SQL server\mssql\data\cpr_attachments.ndf

        3. (DATA DRIVE):\program files\Microsoft SQL server\mssql\data\cpr_log.ldf

      3. Now click OK and the database should start restoring

    5. Changing the DB from a Demo DB to Live DB

      1. Open Query Analyzer
