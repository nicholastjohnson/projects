Configuring HCPCS Codes

10195

Introduction

Adding Administration Codes as HCPCS Procedures

Symptom

Cause

Adding HCPCS Codes

Workaround

Resolution

Adding HCPCS codes through the configuration tab:

  1. Select on the configuration tab
  2. Find Procedures and select Procedures
  3. Change Procedure type (top right) to HCPCS
  4. Select New Procedure
  5. Type in a description that best fits this procedure under Description
  6. Type in CPT code in CPT box
  7. Add units depending on the administration of the drug
  8. Select a category as HCPCS to this procedure
  9. Select on OK

Note: after selecting on OK, this will bring you back to the short list of the procedure screen.  If you desire to assign your specialty to the new procedure you just created, under the search options, find the procedure you just created and select on that procedure.  When a menu comes up, select on Edit Procedure.  This will bring you back to the same screen that you were at to create the procedure.  Towards the bottom of this screen, you will now see a Specialty List (that wasnâ€™t there the first time).  Select on Specialty List.  You can either find your specialty/specialties, or you can select on Set All to highlight all specialties.
