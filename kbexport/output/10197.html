ECG Interface EncounterPRO configuration

10197

Introduction

The only ECG (aka EKG) device that has been tested and proven to workreliably at this point is the Brentwood ECG machine.  Thefollowing instructions assume that you are implementing the Brentwooddevice, but if you are not, the setup for other ECG devices should besimilar.

Symptom


Cause


Workaround


Resolution

  1. Download and install the Brentwood driver 6.2 from JMJ files. (You may have to be logged on as a user with administrative privileges) http://www.jmjtech.com/attachments/drivers/brentwood62.zip

  2. Ensurethat the computer you are installing software on is both a “fat” clientand that it has the correct port format (USB or serial) to correspondto the ECG device.

  3. Once the software is installed, there are two pieces of EncounterPro configuration that need to be performed.
                A. Configure external sources for the computer you are working with, to the
                     the appropriate ECG device

                B. Configure the ECG observation for the appropriate ECG device:

Config of the ECG observation:

 

The results screen of the ECG should contain a simple observation titled “See scanned report.”  When you drill down in this screen you will find a result titled “File Import.”  Configure this simple observation through “Other Options” and switch the “Perform” button to “Attachment.”  You will find in the results box the result “File Import.”  Edit this result and click on the external sources box, select ECG capture device and choose the appropriate device.

 

4.  Plug in the device and make sure that it is turned on.  Now,when you are in the ECG observation, the File import bar will have agreen line on it and you should be able to import the rhythm.  The Brentwoodsoftware has a button that says “Analyze;” when this button isselected, a "snapshot" of the rhythm will be taken.  This"snapshot" will be the image that is saved into EncounterPRO. Select Exit in the midmark interface to get back to EncounterPRO.
