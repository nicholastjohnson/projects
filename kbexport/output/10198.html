Weight missing on drug dosing screen

10198

Introduction

Sometimes the weight does not display on the drug dosing screen, even though there is a weight in the system for that patient.

Symptom

Cause

Some data entry screens add a "Collected" record to an observation.  If a Collected record is added and no subsequent weight value is entered, then the Collected record will be the latest record for the weight observation and EncounterPRO will attempt to display it.  Since the Collected record contains no value or unit, nothing gets displayed on the drug dosing screen.

Workaround

Apply Hotfix 46.

Resolution

This problem is fixed in database modification level 126.
