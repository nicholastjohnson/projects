Setting up Prescriptions to automatically print

10199

Introduction

This article provides information on how to set up automatic printing.

It also provides troubleshooting information if it stops working.

Setting up Prescriptions in VS 4.0

  1. Log into Server with eprojmj account (if User does not exist create user account and make it a domain admin account)
  2. Ensure that printer is added through Printers\Faxes
  3. Open up EncounterPRO and close it. This insert a row in o_computers
  4. dbo_o_Computers

  6. If practice is using a RICOH printer follow "Ricoh Printer Settings"
  7. If practice is using other printer follow "Default Printer Settings"
  8. To set up additional offices that are using RICOH printer follow "Ricoh Additional Offices"
  9. To set up additional offices using other printers follow "Additional Offices"
  10. Open up o_report_printer and ensure that the printer name match what is o_computer_printer
  11. Open up o_server_component and set computer_id for eprojmj which is defined in o_computers.
  12. Start EncounterPRO Service or EncounterPRO Report
  13. Test ordering and ensure that it goes to the printer queue (you may pause printer and then unpause printer to test this)

Setting up Prescriptions in VS 2.0

 

  1. Open up o_server_component and filter for Component type = REPORTSERVER
  2. Log into TECHSERVER with log in of eproreport0001
  3. Open up Printer
  4. Set Default Printer
  5. Set Default Tray to Tray
  6. Restart Report Service
  7. Follow same instructions if practice has multi offices
