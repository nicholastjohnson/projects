Clearing Outlook Cache

10200

Introduction

This article describes how to clear the cache in Outlook.

Symptom

  • New functionality in issues is not present
  • Same issue # keeps showing up when creating new issues
  • Lots of Message Conflicts when saving and closing issues
  • Any Message Conflicts when both parties involved are you

Cause


Workaround


Resolution

In Outlook, while all other issues are closed, go to

  • Tools
  • Options
  • Other
  • Advanced Options
  • Custom Forms
  • Manage Forms
  • Clear Cache
  • Close, OK, OK, OK.


Notes

If Clear Cache doesn't work, you may need to select Customer Form, Clear Cache, Issues, Clear Cache instead of just Clear Cache.
If that doesn't work, try closing and reopening Outlook.
