Setting Prescriptions to Print to 8 1 2 x 11in

10207

Introduction

 

Symptom

Cause

Workaround

Resolution

These are the default settings for setting the prescriptions to 8 1/2 x 11in. However if they do not work you will have to play with the different settings which can be found in the below KB article. Note: Only one prescription will print per page.

http://192.168.155.2/kb/Article.aspx?ArticleNumber=10060 

Version 2.0

Open up o_report_attrubuite

  1. Filter for Report_id = Prescription
  2. Change datawindow.print.paper.source to 2
  3. Change datawindow.print.paper.sizeto 2
  4. If practice is using one line script dataobject should = dw_prescription_size_20
  5. If practice is using two line script dataobject should = dw_prescription2_size_20
  6. Restart License and Report Service
  7. Have users close and reopen EncounterPRO for changes to take effect
  8. dbo_o_Report_Attribute
