Report prints out same patient info

10217

Introduction

This article explains what to do if a practice is complaining of thesame patient's information being printed on all instances of a report.

Symptom

For any patient, when a user previews or prints a report, the same patient's info prints out

Cause

Someone saved changes when closing a report.

Workaround

Per KB Article 10192, if there exists a file in cprfiles/reports/(the report's folder)/(the report filename)_clean.doc, then:

  • Delete the copy of the report file that was changed most recently (the one without _clean at the end of its file name)
  • Remove the _clean suffix from the copy of the report file 
  • Make the file read-only
Else contact a Support or Development person to create a _clean copy of the report file.

Resolution

Inform the practice that they should not save changes to a report file after they are done with it.
