Changing default display of the Problem List

10223

Introduction

"Problem List" is the name of the service that appears when you click on the "Current Status Problems" box on the Summary screen.  It can also be found on several menus.

Symptom

Cause

Workaround

Resolution

If you need to change the way that the Problem List appears from the summary screen, you will need to change the defaults for the service.

Add records as needed to o_service_attribute for service PROBLEMS

Add attribute - assessment_status - with a value of OPEN, CLOSED, or ALL

Add attribute - treatment_status - with a value of OPEN, CLOSED, or ALL

Add attribute - display_mode - with a value of AATT, ATAT, AA, or TT

 

If you need to change the way the Problem List comes up from a menu button, you can add the same records to the appropriate c_menu_item_attribute.item_id values.
