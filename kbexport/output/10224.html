Increase Access Code password - size

10224

Introduction

Symptom

Practice would like to increase access code (password) size.

Cause

Workaround

Resolution

For version 4:
The access code can be changed in  the o_preferences table.

Under o_preferences you would filter in column "preference_id", for "password_length", and then change the access code length to whatever you want.

After changing the length, you should add extra zeros at the end (or begining) of the access codes according to how much you increased it.  This way the user can log on using thier current access code + zeros.  To change user access code go to the "c_user" table.

Ex.- increased access code to 6 digits - original:9876  after change::987600


For version 2:
The access code must be changed in the interface:

Go to Configuration > Preference level=Global, access code length=6

After changing the length, you should add extra zeros at the end (or begining)of the access codes according to how much you increased it.  This waythe user can log on using thier current access code + zeros.  To changeuser access code go to the "c_user" table.
