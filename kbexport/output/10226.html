Checking in a patient changes the date of birth to future date noticed in v2 sites

10226

Introduction

this article explain how the date of birth is changed to future date

Symptom

checking in  a patient (mostly manual check in) changes the birth date to future date

Cause

 

 

a) date_format in o_preferences is not set correct

b) short date format string in "Regional Options" under control panel is not correct.

Workaround

a) set the date_format in o_Preferences to "mm/dd/yyyy"

b) set the short date format in "regional options" under control panel  to "mm/dd/yyyy"
