Invalid column name 'created'

10228

Introduction

Symptom

The below error is receiving when upgrading the mod Level

SQL ERROR = (207) SQLSTATE = 42S22

Microsoft OLE DB Provider for SQL Server

Invalid column name 'created'.

For more information, see Help and Support Center at http://go.microsoft.com/fwlink/events.asp.

Cause

Workaround

Resolution

To resolve this problem the following needs to be done

  1. Change the Modification_level to 120 in c_database_status
  2. dbo_c_Database_Status
    customer_idmajor_releasedatabase_versiondatabase_modedatabase_statusmodification_level
    2116405ProductionOK120
  3. Open up EncounterPRO Server  and select "Check Now"
  4. Upgrade Database to Latest Database Level through EncounterPRO Server
