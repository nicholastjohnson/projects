How to set the Bill flag to DON'T BILL for all procedures except tests and procedures

10229

Introduction

Symptom

Cause

Workaround

Resolution

Run the script below

 

update c_procedure

set default_bill_flag = 'N'

where default_bill_flag = 'Y'

and procedure_category_id in ('XRAY','RCAT','RMRI','RECHO','CHEMISTRY','RNUCLEAR','ULTRA')
