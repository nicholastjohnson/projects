Set labs to not bill

10230

Introduction

Initial install at some sites has the labs set to bill.

 

Symptom

Labs are billing

Cause

The default_bill_flag in c_procedure is set to 'Y'

Workaround

Resolution

Run this sql query to set the labs to not bill.

Note this query will set no billing for XRAY,CT,MRI,Echo,Lab(Chemistry),Nuclear, Ultrasound... remove the procedure_category from the list if you want that type of lab  procedure to bill.

Also, this query will turn off all of the bill flags. So if you want a particular procedure to bill, then must reset using configuration afterward..  

update c_procedure
set default_bill_flag = 'N'
where default_bill_flag = 'Y'
and procedure_category_id in ('XRAY','RCAT','RMRI','RECHO','CHEMISTRY','RNUCLEAR','ULTRA')
