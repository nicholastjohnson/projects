Testing LabCorp Integration

10231

Introduction

For a new release the LabCorp Integration can be tested using the following procedure.

All testors should have access to the testing folder on the gateway machine..Link to this folder copy the text messages from the TestSet sub folder to Watch sub folder.This will cause the BIE Integration Engine route for Testing to process the files in the watch folder.

Then check that the messages have been placed in EncounterPRO Integration Engine (EpIE), which is on wwwserver as Integration db. The table is Message. Check for customer id 999. The messages should have been placed by the BIE integration server here.
