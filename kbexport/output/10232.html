Restoring a patient in Version 2.0

10232

Introduction

This article provides information if a patient entire record has been deleted. This only happen when the record was deleted from p_patient.

Symptom

Cause

Workaround

Resolution

To restore a patient follow the below steps. This can only be done during after hours as it will lock the database for about 15-20 minutes

  1. Backup current database
  2. Locate backup of database that was done prior to the deleltion
  3. Restore backup of database in SQL and name it cpr_restore
  4. Verify patient has no information in p_patient and p_patient_encounter in the live database
  5. Verify patient exists in p_patient in restored database
  6. Open the below script in Query Anzalyer
  7. -- Run script against Live Database
    -- Replace xxxx with the name of the database you are coping from.
    -- Replace yyyy with the cpr_id of the patient you want to copy.
  8. After running it verfiy that patient exisits in live database

Notes
