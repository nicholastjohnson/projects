Package not populating when ordering a new drug

10234

Introduction

Symptom

When you order a new drug - click on the drug, the choose the package and administration, the Screen that comes up next does not include the information that you just entered.

Cause

Wrong default treatment workplan - workplan name contains "Edit RX" instead of "New RX"

Workaround

Resolution

Go to Configuration | Treatment Types | Prescription Drugs  and change the default treatment workplan to "Prescription Drugs, New RX/ Print Last OoO" or "Prescription Drugs, New RX/ Print Now OoO"

Notes
