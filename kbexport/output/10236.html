Making a Cancel Treatment treatment mode

10236

Introduction

Instead of editing an encounter workplan to have a treatment not be ordered, it is often easier to set that treatment's "Custom Trt Mode for QVV" to automatically cancel the treatment as it is ordered.

Symptom

Cause

Workaround

Resolution

Using the example of ROS:

Step 1

Go to Configuration | Treatment Types | ROS | Edit | Treatment Modes  and see if there is already a treatment mode called Cancel or Cancel Now.  If there isn't, click on New Treatment Mode, call it "Cancel Now", and search for a workplan - description contains "cancel" (click on the specialty button if necessary).  Select the workplan.

Step 2

go to Configuration | Preferences | User Defined Properties | Specialties - scroll down to "Custom Trt Mode for QVV ROS" for the encounter type needed and enter "Cancel Now"
