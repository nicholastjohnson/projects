Introduction

This is to explain how or why this popup "patient context changing" message is triggered

Symptom

A popup message appears "Patient context is changing to: "

Cause

“Whenthere are two or more patients in the same room and more than one ofthem has an auto-perform service then it’s possible for the user tofinish a service for one patient and have EncounterPRO automaticallyperform a service for the other patient.  When this happens, theuser might not notice that the patient name in the blue title bar haschanged, so to help try to keep the user from charting data against thewrong patient, EncounterPRO warns the user when the patient contextchanges in between auto-perform services.”
