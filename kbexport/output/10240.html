Creating a Database in MS Access

10240

Introduction

This article describes how to create an Access database for a practice that already has a cpr database in SQL Server.

Symptom


Cause


Workaround


Resolution

Open MS Access.

  1. Under New, click Blank Database. OR goto File, New, Blank Database. Save the database to My Documents as cpr.mdb
  2. In the Tables view, right click and select Link Tables. Under Files of type, select ODBC Databases
  3. If there exists a cpr.dsn under Data Sources, the skip to step #8.  Else, click New
  4. In the next screen select SQL Server, click Next.
  5. Type in cpr, click Next, Finish.
  6. Under Description type in cpr. Select the Server where the database resides. Click Next, Next
  7. Change the default database to cpr. Click Next, Finish, OK.
  8. In the Select Data Source screen, with cpr selected, click OK
  9. Click Save Password, Select All
  10. In the list of tables, deselect any table with the followingname:  c_1*, dt*, jmc*, old*, pbcat*, r*, syn*, sys*, TECH*, tmp*,v_*
  11. Click OK
