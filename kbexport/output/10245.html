Set a workplan to default to a certain E&M level

10245

Introduction

Symptom

Cause

Workaround

Resolution

Add a step in the encounter workplan, New Note, context : Encounter, progress_type : PROPERTY, key : EM_ENCOUNTER_LEVEL and progress text : 3 (or whatever level you want). This creates a record in p_patient_encounter_progress to make it look like that is the user-selected E&M level.
