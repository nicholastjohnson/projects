Services not starting -- no error in EV

10246

Introduction


Symptom

Trying to start the EncoutnerPRO service(s), getting this error:

Cause

For each Cause, see the corresponding Resolution

  1. Mod level is not compatible with the version of EncounterPROServer. Ex EPRO Server 4.05.2.2 is only compatible with mod level 125.
  2. Eprojmj does not have admin rights

Workaround


Resolution

  1.  Bring the mod level to something compatible
  2. Go to Active Directory Users and Computers, go to the Properties of User eprojmj. Click on the Member Of tab. Add Adminstrators.
