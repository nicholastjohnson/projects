Epro Calendar not Functioning

10247

Introduction

Symptom

It freezes, or shows the dates, or information incorrectly , or it is blank

Cause

Workaround

Resolution

1) Repair Encounterpro.

Go to control panel add/remove  programs, look for Encounterpro and change -then repair.

If it is on thin clients  you may want to have everyone logged of  Encounterpro while repairing the calendar.

Should be fine on fat clients

2) If that doesn't work, manually register the calendar control (mscal.ocx) using regsvr32.  The calendar control should be in window\system32.  If this is on a terminal server. make sure you run "CHANGE USER /INSTALL" before registering and "CHANGE USER /EXECUTE" afterwards.

3) If that doesn't work AND MS Access is installed, search for all the instances of the mscal.ocx file.  Delete all copies, run the InstallShield program for MS Access (or MS Office if Access was installed as part of Office), and choose the "Repair" option.  If this is on a terminal server. make sure you run "CHANGE USER /INSTALL" before repairing and "CHANGE USER /EXECUTE" afterwards OR run the installation from Add/Remove programs.
