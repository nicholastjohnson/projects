EncounterPRO Uses of EproLibNet

 

Usage

EproLibNet.Utilities Method

When a TIF file is displayed in a screen control (like a portrait box or scanning/posting preview box) and the c_Attachment_Extension.display_control is set to “Picture” for the “TIF” extension, then EncounterPRO will use EproLibNet to convert the TIF file to a bitmap file for displaying on the screen.

ConvertImage

When EncounterPRO uploads a document to EpIE (EncounterPRO Integration Engine), it uses EproLibNet to make the SOAP call.

putjmjmessage

When EncounterPRO sends a blob to EpIE, it uses EproLibNet to convert the blob to a hex-encoded string.

convertbinarytohex

When a user reviews or transcribes an attachment, EncounterPRO may attempt to open the attachment in a separate process.  If it does, then EncounterPRO will attempt to kill the process when the review/transcribe service exits.  EncounterPRO uses EproLibNet to get the main window handle of the spawned process.

GetMainWindowHandle

EncounterPRO uses EproLibNet to send all messages to the Windows Application Event Log

InitializeEventLog, CloseEventLog, LogEvent

EncounterPRO uses EproLibNet to read files into blob variables.  One example is importing attachments.

FileToBytes

EncounterPRO uses EproLibNet to write blob variables to files.  This is used anytime an attachment is displayed or edited.

BytesToFile

EncounterPRO uses EproLibNet to encrypt and decrypt preference values which should not be visible to the user.  One example is the “Upgrade FTP User/Pwd” Computer Preferences.

Encryptstring, decryptstring

EncounterPRO uses EproLibNet to transform an XML file using an XSLT file.

TransformXML

EncounterPRO uses EproLibNet to copy files from an FTP site.  This is used for downloading new versions of EncounterPRO and EproLibNet.

CopyFTPToUNC

 


 

EncounterPRO Uses of EproLib4

 

Usage

EproLibNet.Utilities Method

EncounterPRO uses EproLib4 to get the screen resolution.

screen_resolution_x, screen_resolution_y

EncounterPRO uses EproLib4 to convert a numeric value into words for displaying the dispense amount on prescriptions.

number_to_text

EncounterPRO uses EproLib4 to get the printer resolution for calculating certain report features.

printer_resolution_x, printer_resolution_y

EncounterPRO uses EproLib4 to allow the user to select a color.

select_color

EncounterPRO uses EproLib4 to render a penop signature file into a bitmap for printing on a report.

Render_bmp

EncounterPRO uses EproLib4 to activate offsite (i.e display) a file using an ActiveX control.

display_ole_file

EncounterPRO uses EproLib4 to capture an audio file from a standard audio device (i.e. microphone)

record_audio

EncounterPRO uses EproLib4 to get an image file from a floppy recorded by a Sony Mavica digital camera.

Get_image

EncounterPRO uses EproLib4 to capture a signature from a tablet device using the Penop signature capture software.

Get_signature
