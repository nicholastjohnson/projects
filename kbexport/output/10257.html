Error ordering encounter workplan Encounter Workplan not found

10257

Introduction

Symptom

Error Message:

4.05.33.7 (10/8/2004) >>> u_patient - (new_encounter()) Error ordering encounter workplan
Jenny: 4.05.33.7 (10/8/2004) >>> u_sqlca - (check()) SQL ERROR = (50000) SQLSTATE = 42000 
Microsoft OLE DB Provider for SQL Server
Encounter Workplan not found (Test2, 9384)

Cause

The most likely cause is that the patient being checked in did not fit the age criteria for the workplan(s) attached to the encounter being created.

Workaround

Resolution

Remove the age range criteria from

  • Configuration
  • Encounter Types
  • Edit encounter type that is having the problem
  • Click on the workplan under Workplan Selection
  • Edit Criteria
  • Click [clear] next to Patient Stage.
