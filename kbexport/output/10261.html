Column does not allow nulls

10261

Introduction

Symptom

Upgrading Database Level or Config Level throws one of the below errors

SQL ERROR = (515) SQLSTATE = 23000

Microsoft OLE DB Provider for SQL Server

Cannot insert the value NULL into column 'bill_observation_perform', table 'cpr_40.dbo.c_Treatment_Type'; column does not allow nulls. INSERT fails

OR

SQL ERROR = (207) SQLSTATE = 42S22

Microsoft OLE DB Provider for SQL Server

Invalid column name 'created'.

Cause

Workaround

Resolution

Run this in Query

sp_rebuild_constraints 'c_Treatment_type'

Click Upgrade button again. You do not need to restore the database and start over.
