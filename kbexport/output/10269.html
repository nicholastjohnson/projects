Introduction

This article explains what's included in patient summary report

Symptom

a) Summary report not showing all the vitals / procedures /histories /assessments

Cause

a) There's an maximum interval attribute and that needs to be set to the required interval(default 1 year) by customer to include all the results

b) There are limitations in maximum no. of rows displayed (default to 20 records) with encounters, vitals and objectives. Make sure they are set to maximum size.

c) Report displays only OPEN assessments. So if the customer complain it's not showing all then tell them it shows only open assessments for that patient from beginning to recent encounter date.
