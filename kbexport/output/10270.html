Notes

This report resembles the FL WIC report given to us by Island Coast Peds. This report can be installed from ftp://ftp.jmjtech.com/EP_Reports/4.0.

After the report is installed, it should be configured in EncounterPRO. The following fields will have to be filled out by the user at runtime:

  • All checkboxes, and any fields dependent on the status of a checkbox
  • Date of WIC Certification Appointment
  • Birth weight
