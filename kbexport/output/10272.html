How to rename a Database

10272

Introduction

This provides instructions on how to rename a database.

Symptom

Cause

Workaround

Resolution

Follow the below instructions

  1. Ensure that there are no open sessions of EncounterPRO, you can check this by going to Enterprise Manager\Management\Process Info. If there are any sessions connected to the database; right click on them and select kill process. Close Enterprise Mananger
  2. Open up Query and run the following query against the master database. The database names will have to be changed accordingly
  3. sp_renamedb 'olddatabase','newdatabase'
