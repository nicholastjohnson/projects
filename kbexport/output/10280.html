Restoring a database DB

10280

Introduction

These instructions will guide you through restoring a Database - DB.

Symptom

Cause

Workaround

Resolution

These instructions are assuming that you have already got a backup file located.  Backup files end with a .BAK extension.

1.  First thing you must do is create a new database

a.  right click on databases and select new Database...

          

b.  Type the name of the new database in the name field.

c.  Select OK

d.  Right Click on the new database that you created

e.  Select all tasks and restore database

. 

f.  Click on the from device radio button

g.  then click on the Select Devices... button

h.  click on the Add... button

i.  click on the ... button

j.  browse to the location of the .bak file and select OK button.

k.  Click OK till you are back at the General tab and select the Options tab

l.  Select the Force restore check box.

m.  make sure that teh move to physical file name boxes have the right location and filename for the output of the restore. (This is very impotant if you are adding another copy of the database.  You do not want to right over the database that is already in there.)

n.  Click OK once all information has been entered. 

2.  This process can take a really long time depending on the size of the database that you are restoring.
