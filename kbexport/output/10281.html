Patient Attachment does not show in EncounterPRO

10281

Introduction

Symptom

The file exisits in the database and shows on the category tab (example Lab Reports - 2), but there is no bar to display the attachment.

Cause

This happens when the USER_ID that posted this attachment is missing from C_USER.

Workaround

Resolution

Add a record to c_user that matches the user_id in p_attachment
