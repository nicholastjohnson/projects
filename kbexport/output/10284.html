Creating Restart License Job in Version 2.0

10284

Introduction

This article provides information on what to do if the license service seems to start successfully, but it still throws a licensing error when EncounterPRO is opened

Symptom

Cause

Workaround

Resolution

  1. Stop and disable license server
  2. Create following job in SQL
    • Open up Enterprise Manager
    • Select Tools
    • Select Job Scheduling
    • Select Next
    • Select Transact-SQL command
    • Select Next
    • Select Database name usually cpr
    • Enter this file in this box
    • Select Next
    • Select "On a Recurring basis"
    • Select "Schedule"
    • Select "Daily" and "Occurs every 1 minute"
    • Select "Ok"
    • Select Next and Next again
    • Rename Job "EPRO Restart Service"
    • Select Finish and OK
  3. Open up EncounterPRO and ensure that their is no error msg and you can create a new encounter
