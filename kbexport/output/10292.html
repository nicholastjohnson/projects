Unable to view chart correctly on XP machine

10292

Introduction

Symptom

When you open the chart it looks similiar to the below screen or you get this error

Unknown Caller - (SYSTEM ERROR) errornum:39

message :Error accessing external object property selectionrectangle at line 1 in constructor event of object ole_image of u_letters.

where :u_letters

object :ole_image

event :constructor

line :1

Cause

Workaround

Resolution

1)      Locate the  xp imaging.zip which can be found on our CD \Misc. Instructions\Programs

2)      Unzip to “C:/Windows/System32” (Depending on the name of the operating system it may be “C:/Winnt/System32” or in rare instance when there is no folder called System32 copy it in the folder called System

3)      Click on the start button and go to run and type in cmd

4)      Type in cd \ and select “Enter”

5)      Type in cd windows (in some cases it will WINNT, depends on your operating system) and select “Enter”

6)      Type in cd system32 (if your computer has no directory by this name type in System and select “Enter”

7)     Type in regsvr32 space imgedit.ocx and press the “Enter” key

8)      Make sure you get a succeeded msg
