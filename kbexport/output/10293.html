EPRO Service will not start

10293

Introduction

Symptom

The below error msg occurs when you start EncounterPRO Service

System.Data.SqlClient.SqlException: Procedure 'sp_scheduled_services' expects parameter '@ps_user_id', which was not supplied.

at System.Data.SqlClient.SqlCommand.ExecuteReader(CommandBehavior cmdBehavior, RunBehavior runBehavior, Boolean returnStream)

at System.Data.SqlClient.SqlCommand.ExecuteReader()

at EncounterPROService.ServiceThreadManager.MainServiceLoop()

Cause

Workaround

Resolution

Delete the 121 scripts with this query:

DELETE FROM c_Database_Script

WHERE script_type = 'Database'

AND modification_level = 121

GO

Then set the database mod level back to 120, and re-upgrade them to 125.
