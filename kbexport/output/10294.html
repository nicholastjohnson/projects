Inactive users on user short lists

10294

Introduction

Symptom

When a user is inactivated, the user remains on the user short list.

Cause

When a user is inactivated, the user_status field in the database is changed to 'NA'.  The top 20 table and the user short list screens are not checking to see if the users on the short lists are active.

Workaround

Apply hotfix 63.

Resolution

This problem is fixed in database mod level 127.
