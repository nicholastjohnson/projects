Wait Service waits indefinately

10296

Introduction

 

Symptom

The wait service is configured to automatically complete itself after a specified amount of time, but it waits indefinately.

Cause

The wait service is automatically completed by the appropriate JMJ Server Service, so the server service must be running.  Also, in Mod Level 121 the wait service stopped supporting seconds as the time unit.

Workaround

Make sure the JMJ Server Service is running

Apply Hotfix 64 to fix the problem with the time units in seconds.

Resolution

This problem is resolved in Mod Level 127.
