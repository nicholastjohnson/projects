DB Scripts fails for mod level 126

10301

Introduction

Symptom

Running DB Scripts for mod level 126 fails with an error message such as:

SQL ERROR = (208) SQLSTATE = 42S02
Microsoft OLE DB Provider for SQL Server
Invalid object name 'dbo.p_Patient_Alias'.

Cause

Mod level 126 was re-released to testing after the initial release of mod level 127 to testing.  This caused several trigger objects modified in mod level 127 to be included in the compilation of DB_Scripts.

Workaround

DB Scripts for mod level 126 has been removed from the Utilities tab.  If you need to run DB Scripts for mod level 126 then get it from the FTP site and execute it in Query Analyzer:

ftp://ftp.jmjtech.com/EP_Quick_Fixes/DB Scripts 126.sql

 

Resolution

This is a known problem with the release manager and will be fixed in a future upgrade of the release manager.
