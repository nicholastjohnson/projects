Epro closes when viewing unposted attachments

10302

Introduction

This article explains what to do when view unposted attachments closes EncounterPRO

Symptom

Going to Utilities, clicking on Scan/Post will shutdown EncouterPRO with an error like this:

Event Type:    Information
Event Source:    Application Error
Event Category:    (100)
Event ID:    1004
Description:
Reporting queued error: faulting application encounterpro.exe, version4.5.32.24, faulting module ntdll.dll, version 5.2.3790.0, fault address0x0000207c.

Cause

Bad attachment file in queue

Workaround

Go to p_attachment, filter for cpr_id is null. If there is one where status = OK, set its status to NA.

Resolution

Don't import bad attachment files.
