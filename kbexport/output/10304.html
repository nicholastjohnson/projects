If the progress type is not designated in the RTF command, then whether or not the progress note prints is determined by the [display_flag] value in the appropriate progress type definition table.  The flag can come from the type-specific table (c_Assessment_Type_Progress_Type in this case) or the generic table, c_Object_Default_Progress_Type.  If the progress type is not found in either table, the display flag is assumed to be ‘N’.
