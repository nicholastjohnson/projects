Resolution

In order to make the object-level confidentiality functions accessible, we use buttons to add progress notes.  These buttons create a Progress Note with the context object you want to make confidential (patient, encounter, DX, treatment) and the progress note type of Confidential and the subject needs to be Deny or Allow (deny to restrict and allow to un-restrict). I would think that the patient-level button would go on the Patient Demographics screen, the Encounter-level button on the SOAP screen, other items; the DX and treatment buttons on their respective dashboard screens.
