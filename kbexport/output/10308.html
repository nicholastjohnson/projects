How to enable the ICD CPT descriptions for DFT^PO3 messages

10308

Introduction

This article explains how to enable or dsiable the ICD / CPT descriptions in EncounterPRO charge export interface.

Symptom

a) Some PM Systems (like Elligence) fail to parse chargemessage if the descriptions are enabled and in result of that thecharge posting failed in PM import system.

b) Some PM Systems (like PCC) request us to send the descriptions in the charge messages.

Cause


Workaround


Resolution

Enable descriptions for ICD & CPT in HL7 DFT^P03 (charge) message:

Add this attribute show_cpt_icd_description = "Y" for thecomponent_id = “TRN_OUT_FILECOPY_HL7” to c_Component_Attributetable(this is a generic out going component name. for 2PMS vs 1EPRO dbit could be different. To see the component_id’s for outgoing check theo_message_subscription table for direction=”O”].

Disable descriptions for ICD & CPT in HL7 DFT^P03 (charge) message:

Add to c_Component_Attributetable this attribute: show_cpt_icd_description = "N" for thecomponent_id = “TRN_OUT_FILECOPY_HL7”.  (This is a generic out going component name. For 2 PM systems integrated with 1 EPRO db,it could be different. To see the component_id for the outgoing component, check theo_message_subscription table where direction=”O”.)
