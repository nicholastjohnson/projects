Image Viewer does not display

10309

Introduction


Symptom

The EPImageViewer does not open when you try to view an image attachment, and no error is presented.

In the Event Viewer you find an error like this:

Ver. NA >>> Atalasoft.dotImage - (Initialize) Error Loading Image in file c:\temp\jmj_204_8001.0__050621122330_4.tif

System.DllNotFoundException: Unable to load DLL (AtalaImaging).

Cause


Workaround

Resolution

Locate the AtalaImaging.dll file, which should be in C:\Program Files\Common Files\JMJTech Common\EproLibNET.

Copy the AtalaImaging.dll file to the System32 folder.  Thiswill usually be either C:\WINNT\System32 or C:\WINDOWS\System32.
