HOWTO Get a log file from Setup.exe

10312

Introduction

Sometimes a setup fails without an informative error.  In these cases, a setup log is needed in order for support personnel or a developer to determine the problem.

Symptom

Cause

Workaround

Resolution

To log an installation, one must run Setup.exe with specific command-line parameters to generate a log file.  You can do this by creating a shortcut and adding the parameters to the path, or you can run setup from the command prompt.

This is an example of how to tell Setup.exe to generate a log file:
setup.exe /v"/l*v C:\Setup.log"

Once the setup has completed, collect the log file--C:\Setup.log in this case--and use it for further troubleshooting or escalate the issue if necessary with the setup log as an attachment to the issue.
