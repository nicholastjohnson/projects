User cannot add new procedure

10315

Introduction

This article will explain how to fix a problem in version 4.05.33.7 where a user cannot add a new procedure.

Symptom

The below errror appears in the event viewer when trying to add a new procedure.

4.05.33.7 (10/8/04) >>> u_ds_data - (dberror) DATAWINDOW SQL ERROR = (8114) Select error: SQLSTATE = 22018

 Microsoft OLE DB Provider for SQL Server

 Error converting data type nvarchar to decimal.

 Dataobject = dw_sp_new_procedure

 SQL Statement = 1 execute dbo.sp_new_procedure   @ps_procedure_type = 'PROCEDURE', @ps_cpt_code = '123', @pdc_charge = NULL, @ps_procedure_category_id = 'ANESTH', @ps_description = 'test', @ps_service = NULL, @ps_vaccine_id = NULL, @pr_units = 1, @ps_modifier = NULL, @ps_other_modifiers = NULL, @ps_billing_id = NULL, @ps_location_domain = NULL, @pi_risk_level = 0, @ps_default_bill_flag = 'Y'

Cause

Typo in hotfix 65 caused this problem. 

Workaround

Resolution

Re-run hotfix 66.
