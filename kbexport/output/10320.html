Turn on Billing Interface OR How to cancel all the billing requests for a given date range

10320

Introduction

This article explains how to cancel the billing requests for a given date range as well as the steps needs to be followed before activating the billing interface.

Symptom

Billing interface has never been turned on or had problems and stopped working. For any of these cases, always CANCEL ALL THE PREVIOUS BILLS BEFORE STARTING THE BILLING INTERFACE

Cause


Workaround


Resolution

HOW TO CANCEL QUEUED BILLING REQUESTS FOR A GIVEN DATE RANGE:

RUN THE QUERY DURING THE PRACTICE OFF-HOURS

VERSION 2

  1. Run the below query

Update p_Patient_Encounter

set billing_posted='Y'

where encounter_date <  [enter the date in single quotes] and bill_flag = 'Y' and billing_posted = 'N'

 

Example: to cancel all the bills prior to '6/30/2005' then it'll be something like where encounter_date < '6/30/2005'

  1. Set EncounterPROBilling and EncounterPROTransportOut Status to OK in o_server_component
  2. Restart EncounterPROBilling and EncounterPRO TransportOut Services

VERSION 4

  1. Run the below query

Update p_Patient_WP_Item

set status = 'CANCELLED'

WHERE dispatch_date < [enter the date in single quotes] 

AND ordered_service in ('SENDBILLING','RETRYPOSTING')

AND status in ('DISPATCHED','STARTED')

 

Example: to cancel all the bills prior to '6/30/2005' then it'll be something like where dispatch_date < '6/30/2005'

  1. Set EncounterPROBilling and EncounterPROTransportOut Status to OK in o_server_component
  2. Restart EncounterPROService Services (or EncounterPROBilling and EncounterPRO TransportOut)
