Get patients for a given date of birth range

10322

Introduction

this article explains how to get a list of patient records for a given date of brith range

Symptom

Cause

Workaround

Resolution

Run the following query by filling in your requested date of birth range. the below query returns the results of patients born between year 2000 to year 2001.

Query:

select billing_id,first_name,last_name,date_of_birth from p_patient where date_of_birth between convert(datetime,'1/1/2000',101) and convert(datetime,'12/31/2001',101) order by date_of_birth
