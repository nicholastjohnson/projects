Error on Printing the Show Data under development tab

10323

Introduction

this article explains why the "show data" button reports an error while printing the details.

Symptom

errornum:39
message :Error accessing external object property text (Line 1 Column 40: incorrect syntax.) at line 3 in clicked event of object cb_print of u_cpr_page_growth_percentiles.
where :u_cpr_page_growth_percentiles
object :cb_print
event :clicked
line :3

Cause

After printing the vital details, Epro prints the patient name at the bottom of the page. this includes firstname, lastname, middlename and nickname. if any of them have double quotes it'll then report the above error(look in symptoms).

Workaround

Use the EncounterPRO Server DB Maintenance application to run the "Clean Up Double Quotes In Patient Names.sql" script from the Utility Scripts tab.

Resolution

This problem will be fixed in EncounterPRO Client version 4.05.35.
