Setting up User Manual and Help Preferences

10327

Introduction

This article explains how to set the user manual preference or if a user is unable to view the user manual or Help Preference

Symptom


Cause


Workaround


Resolution

    User Manual

    1. Select the Configuration tab.
    2. Select Preferences.
    3. Select Computer Configuration (on the left).
    4. Select User Manual Location.
    5. Paste the following link: http://www.jmjtech.com/attachments/Manualv4_0/epro_40_user_manual.pdf
    6. Select Finished.
    7. Select OK
    Help File Prefix
    1. Select the Configuration tab.
    2. Select Preferences.
    3. Select Computer Configuration (on the left).
    4. Select Help File Prefix.
    5. Paste the following link http://www.jmjtech.com/attachments/online_help_4x/
    6. Select Finished.
    7. Select OK
    Ifa practice does not have internet turned on for all users and computers you will need to downloadthe entire epro_4x_online_help folder from the FTP site and share thefolder (read only) and change the preference to the following

    dbo_o_Preferences
    preference_typepreference_levelpreference_keypreference_idpreference_value
    SYSTEMGlobalGlobalHelp File Prefix\\ServerName\online_help_4x\
    SYSTEMGlobalGlobalUser Manual Location\\ServerName\online_help_4x\epro_40_user_manual.pdf
