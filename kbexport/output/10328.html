Config Mode

10328

Introduction

This article explains how to put EncouterPRO in or out of Config Mode

Symptom


Cause


Workaround


Resolution

In EncounterPRO, go to Configuration, Preferences,
Preference Type = General Preferences,
Level = Computer,
Preference = 000Standard Configuration Mode
Set this to Yes to turn it on.
Set this to No to turn it off.
