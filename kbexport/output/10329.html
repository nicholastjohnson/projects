Patients going to the wrong waiting room

10329

Introduction

This articles tells what to do if patients are checking in to the wrong waiting room.

Symptom

Patients check in for a Well Visit and goes to the Sick waiting room, or vice versa.

Cause

GET PATIENT step in a workplan is set to Any Waiting Room and should be set to the specific waiting room.

Workaround

Set the GET PATIENT step of the workplan correctly:

  1. Go to Config, Encounter Types
  2. Find the workplans associated with this encounter type
  3. Go to Config, Workplans
  4. Set Workplan Type to Encounter,
  5. For each workplan associated with this encounter type:
    1. Display/Edit this workplan
    2. Select Edit Workplan
    3. Get Patient, Room Assignment, <Default Room Type>, set to the practice's name
    4. Set the specific room
