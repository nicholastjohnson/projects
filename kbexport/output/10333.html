Transport In and Out won't start

10333

Introduction


Symptom

Below are the errors that might appear in the Event Viewer

4.05.33.7 (10/8/2004) >>>u_component_incoming_filecopy_hl7 - (ahc_connect) ERROR: connection toAHC Messenger object failed.

4.05.33.7 (10/8/2004) >>> u_component_incoming_filecopy_hl7 - (xx_initialize) Unable to connect to AHC Interface..

4.05.2.9 (12/16/2005) <Server PB Code> >>>u_component_serverservice_incoming - (xx_initialize()) Unable to getcomponent (TRN_IN_FILECOPY_HL7)


Cause

The DLL's need to be re-registered

Workaround


Resolution

Stop all Services and all encounterpro.exe applications from Task Manager
Download HL7Mapping folder from the FTPsite.\EP_Integrations\HL7Mapping (If this folder already exists, justdownload the registerahcdlls.bat into the HL7Mapping folder.
Run "registerahcdlls.bat"
Start Encounter Services
Ensure that their are no errors in the Event Viewer
