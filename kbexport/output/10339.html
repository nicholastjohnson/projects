Editing a License

10339

Introduction

 

Symptom

Edit License

Cause

Workaround

Resolution

1.  Open > elicense

2. select the practice and click "edit license".

3. Make the required changes to the license and then choose "update".

4. Insert Floppy Didk into the A: drive  click "ok" to create disk.

5. Go to the ftp site, under "cprfiles" and locate the file "jmjlic", and rename it->(suggest,"jmjlicold").

6. Copy this newly created license to the ftp site. Download the file to the "cprfiles" folder.

7. Now rename the license file you just downloaded to "jmjlic".

8. Restart the license service for the practice you updated.
