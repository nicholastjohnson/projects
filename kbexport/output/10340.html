Resolution

 Follow the below steps

  • Go to Public Folders\Issue Tracking\Issues
  • Right Click on Issues and Add to Favorites
  • Go to Favorites and right Click on Issues and select Add to Other Tasks
  • Click on Tasks and then Issues
  • Select New Issue
Task Tab
Required Fields

  • Ensure that a Issue # has been assigned
  • Enter Description in Blank box
  • Click on Contacts and link Client


Issue Status Tab
Required Fields (This can also be used for modifying an existing issue)

Assign Current JMJ Owner. All support issues assign to Support
Enter in Contact Name
Enter in Email Address
Check appropriate check boxes

If Issue is Urgent sent SR to appropiate owner

Closing Issue

Enter in comments that issue is resolved

Inform Client (Support will only notify clients on behalf of Development)

Change Issue Status to Closed

Select Complete button or Change Status to Completed or % Complete to 100%
