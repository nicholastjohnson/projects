Use this query to find out how many records queued up for a given user for "supervisor review"

select * from p_patient_wp_item where owned_by='0001user' and status in ('dispatched','started') and description like '%review by supervisor%'

Use this query to complete (close) them for a given user:

Update p_patient_wp_item

Set status = 'Completed'

Where owned_by='0001user' and status in ('dispatched','started') and description like '%review by supervisor%'
