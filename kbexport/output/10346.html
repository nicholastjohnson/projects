Setting up a Service

10346

Introduction

Symptom

One wants to add or remove a service

Cause

Workaround

Resolution

1. Open a command prompt window

 

2. In the command prompt type instsrv <service name> <exe location>

Ex.-

Instsrv EncounterProBilling C:\Mydir\encunterpro.exe

 

3. Once the service is created go to “o_server_component” and put the new service in the “service_name” column=>Make sure that the service you created is in accordance with the system_user_id logically

Once the service is created go to Administrative Tools, then Services.

Set the password to !$Epro&&

Set it to Automatic and start the Job.

To Remvoe the service  go t command prompt and type instsrv remove <service name>
