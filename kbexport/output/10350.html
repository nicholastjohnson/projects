Ordering certain item s from treatment list bills strange cpt codes

10350

Introduction

Symptom

Examples:

An encounter instruction ordered from a treatment list has an ultrasound code, or a flu shot attached to it.

A prescription medication might have an IM injection admin billed to it.

 

Cause

The u_assessment_treatment_definition for the treatment has an inappropriate attribute associated with it.

Workaround

Resolution

Search for the description of the treatment in u_assessment_treatment_definition.description.

Search for that id in u_assmt_treat_def_attrib and delete any records with attribute "procedure_id"

Notes

This is the SQL view of the ACCESS query that I put together to see if there are any procedure ids attached to some inappropriate treatment types.

 

SELECT dbo_u_assessment_treat_definition.treatment_type, dbo_u_assessment_treat_definition.treatment_description, dbo_u_assessment_treat_def_attrib.attribute, dbo_u_assessment_treat_def_attrib.value
FROM dbo_u_assessment_treat_definition INNER JOIN dbo_u_assessment_treat_def_attrib ON dbo_u_assessment_treat_definition.definition_id = dbo_u_assessment_treat_def_attrib.definition_id
WHERE (((dbo_u_assessment_treat_definition.treatment_type)="instruction" Or (dbo_u_assessment_treat_definition.treatment_type)="medication" Or (dbo_u_assessment_treat_definition.treatment_type)="activity" Or (dbo_u_assessment_treat_definition.treatment_type)="filenote" Or (dbo_u_assessment_treat_definition.treatment_type)="followup" Or (dbo_u_assessment_treat_definition.treatment_type)="PRACTIVITY") AND ((dbo_u_assessment_treat_def_attrib.attribute)="procedure_id"));
