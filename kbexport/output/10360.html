Reading the Database Script Log table

10360

Introduction

Every script in c_Database_Script which is executed by EncounterPRO is logged in the c_Database_Script_Log table.  It can be usefull sometimes to be able to see what scripts have been run and when.

Symptom

Cause

Workaround

-- This query displays all of the database scripts which have been executed
-- in this database.

SELECT l.script_id,
 l.executed_date_time,
 s.script_type,
 s.description ,
 s.modification_level,
 l.completion_status
FROM c_Database_Script_Log l
 INNER JOIN c_Database_Script s
 ON l.script_id = s.script_id
ORDER BY l.executed_date_time desc
