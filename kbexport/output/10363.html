Unable to create temp folder

10363

Introduction

This article details how to deal with this error from TransportOut:
u_component_outgoing_filecopy_hl7 - (send_file()) unable to create temp folder c:\temp.

Symptom

  1. Getting this error in Event Viewer every few seconds: 
    1. u_component_outgoing_filecopy_hl7 - (send_file()) unable to create temp folder c:\temp.
  2. Everything seems to be propoerly configured in c_component_attribute and o_service_component
  3. Practice has no C:\ drive

Cause

Versions 33.7 and earlier are looking for the temp directory attribute(temp_attribute in c_component_attribute table) otherwise hard coded touse c:\temp.

Workaround


Resolution

Add a row to c_component_attribute where

  • component_id=TRN_OUT_FILECOPY_HL7,
  • attribute_sequence is automatically assigned,
  • attribute=temp_directory
  • value=N:\TEMP or whatever is the path of the available drive with a temp directory
