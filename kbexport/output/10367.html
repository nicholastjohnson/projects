How to change the Brand Necessary phrase on a prescription

10367

Introduction

When the user sets the "Brand Name Required" setting to "Yes" for a prescription drug, then EncounterPRO will append the phrase "Brand Necessary" to the drug sig on the printed prescription.  If the user desires a different phrase to represent Brand Name Required = Yes, the desired phrase can be set in o_Preferences.

Symptom

Cause

Workaround

Resolution

Add a record to o_Preferences like this, but with the desired phrase in the preference_value column:

 

dbo_o_Preferences
preference_typepreference_levelpreference_keypreference_idpreference_value
PREFERENCESGlobalGlobalbrand_necessary_phraseBrand Medically Necessary
