Configuring the Export Chart menu button

10369

Introduction

New feature in EncounterPRO client 34.7, the Export Chart Service will need to be configured if using with a configuration version earlier than version 24. Configuration version 24 will have the required button already added.

The Chart Export service is a patient-level service which exports an entire patient chart, including attachments, into a set of files in a specified directory.

Symptom

Cause

Workaround

It is recommended that a button to call the Export Chart service be added to the Patient Demographics screen. You can addd a button for this service to any menu or include it as a workplan item, as desired.

To add the Export Chart service to the Patient Demographics screen:

1. First, turn on configuration mode.

    a.  Log in to EncounterPRO.

    b. Select Configuration.

    c. Select Preferences.

    d. Select General Preferences on the left and Computer at the bottom of the screen.

    e. Locate the preference, "000 Standard Configuration Mode" and select that bar. Select Yes, then Finished.

    f.  Select OK, then Exit to return to the Office View.

2. Select the Charts tab. Enter the name of a patient, preferably a Test patient. Select the patient from the list. The Patient Demographics screen appears. Notice the small text above the menu at the bottom of the screen. This is the menu name.

3. Click on the menu name. The Menu Properties screen appears.

4. Select Create Local Copy. A new bar will appear in the upper right, labeled Local Version. Be sure this bar is highlighted, then select Edit Menu.

5. Select New (to add a new button), Service, Patient. A list of patient-level services appears.

6. Select Chart Export. There are no parameters to define, select Finished.

7. You can change the text of the menu button, if desired. Select Finished, OK, then Finished. The Patient Demographics screen reappears.

8. If your menu had more buttons than could fit on the screen, the right-most button on the menu will be Other Options. If you do not see your new button on the Patient Demographics screen, select the Other Options button to reveal the remaining buttons.

9. If you are done configuring menus, repeat step 1, changing the preference to No. If you leave your workstation in Configuration Mode, it may slow down the display.

Resolution

Notes

The Chart Export function allows the user to specify the folder on the workstation or network where the files will be stored. After the export service completes, the user can copy that entire folder to a CD-ROM or other media.
