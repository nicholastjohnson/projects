Configuring the Uncancel Encounter menu

10370

Introduction

New feature in EncounterPRO client 34.7, the Uncancel Encounter service will need to be configured if using with a configuration version earlier than version 24. Configuration version 24 will have the required button/menu already added.

The Uncancel Encounter service is restricted to those users with the SuperUsers, priviledges and those specifically granted access to that service under the Authorize Services function.

 

Symptom

Cause

Workaround

It is recommended that a button to call the Uncancel Encounter service be added to the Patient Encounters screen (accessed from the Patient Demographics screen). You can addd a button for this service to any menu where you can select or view the canceled encounter.

To add the Uncancel service to the Patient Encounters menu:

1. First, turn on configuration mode.

    a.  Log in to EncounterPRO.

    b. Select Configuration.

    c. Select Preferences.

    d. Select General Preferences on the left and Computer at the bottom of the screen.

    e. Locate the preference, "000 Standard Configuration Mode" and select that bar. Select Yes, then Finished.

    f.  Select OK, then Exit to return to the Office View.

2. Select the Charts tab. Enter the name of a patient, preferably a Test patient with a canceled encounter. Select the patient from the list. The Patient Demographics screen appears.

3. Select the Encounters tab. Thie tab shows All Encounters, Open Encounters, and Canceled Encounters.

4. Select the Canceled button, then select one of the canceled encounters. Notice the small text above the menu at the top of the screen. This is the menu name.

5. Click on the menu name. The Menu Properties screen appears.

6. Select Create Local Copy. A new bar will appear in the upper right, labeled Local Version. Be sure this bar is highlighted, then select Edit Menu.

7. Select New (to add a new button), Service, Encounter. A list of encounter-level services appears.

8. Select Uncancel Encounter. There are no parameters to define, select Finished.

9. You can change the text of the menu button, if desired. Select Finished, OK, then Finished. The Patient Encounters screen and menu reappears. The new button will not appear until you refresh the display. Select Cancel, then select one of the canceled encounters. The menu reappears with the new Uncancel button.

10. If you are done configuring menus, repeat step 1, changing the preference to No. If you leave your workstation in Configuration Mode, it may slow down the display.
