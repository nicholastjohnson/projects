Medical Manager error no filenames are available

10372

Introduction

This article explains what to do if you get this error

Symptom

  1. In Medical Manager, the practice is getting this error:
    1. No filenames are available, please purge export directory
  2. Pts are not checking in from and/or bills are not posting to Medical Manager



Cause

The pmshare directory is full of all the possible names for files created by Medical Manager.
This usually happens when the EncounterPRO service(s) stop for some reason.

Workaround

Start the EncounterPRO Services

Resolution

Install a version of the EncounterPRO services that doesn't stop unexpectedly.
