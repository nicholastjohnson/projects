New pts not checking in

10373

Introduction

This article explains what to do when new patients won't check in from a PM system after recently upgrading Epro Server or some other EncounterPRO product.

Symptom

An error like this in EV:

Event Source: EncounterPROMsgHandler
4.05.2.4 (4/22/2005) >>> u_component_serverservice_messages - (timer) Error in timer_ding function (Bad runtime function reference at line 22 in function timer_ding of object u_component_serverservice_messages.).

Cause

Old interface pbd

Workaround

Copy the correct pbd from the techserv (look in ftp under version 4.0\eproserver) common folder
