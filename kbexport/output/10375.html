JMJTech missing login name

10375

Introduction

This article explains how to correct the rare case where the login name for jmjtech is blank

Symptom

In Enterprise Manager, under a db, Users, there is a user where name=jmjtech and Login Name is blank

Cause


Workaround

Run this in Query Analyzer:
sp_dropuser jmjtech

Next try to add the user in the Users screen, including the role cprsystem.
If you can this error:
Error 15007: The login 'jmjtech' does not exist
it is because the Login jmjtech has not been added Security, Logins.
Add this login, then try adding the user again
