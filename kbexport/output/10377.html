How to Export All Patients to EncounterPro

The first time you enable the interface between elligence and EncounterPro, you may wish to have all patient accounts in elligence sent to EncounterPro in order to populate that database. Follow these steps to make that happen:

1- Manually create an empty file in the following folder with the name A04.dump:

C:\Program Files\DSI\elligence Application Server\website\Bin\Global\HL7\ {database name}\JMJ\Out

2- When the system finds that empty file in that folder it will know to begin exporting all patient accounts to EncounterPro, using the settings from the elligence Interface Manager. Please contact JMJ EncounterPro's Technical Support for help with using the data from this point forward.
