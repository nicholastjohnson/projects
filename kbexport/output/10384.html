Attending physician report for epro v2

10384

Introduction

This article explains how to use a report in MS Access that gives usersa list of the attending provider for patients that had directencounters in a given date range. This report is for users onEncounterPRO client version 2

Symptom


Cause


Workaround


Resolution

Here is the query:

SELECT dbo_c_User.user_full_name, dbo_p_Patient.billing_id,dbo_p_Patient.first_name, dbo_p_Patient.last_name,dbo_p_Patient_Encounter.encounter_date
FROM (dbo_p_Patient_Encounter INNER JOIN dbo_p_Patient ONdbo_p_Patient_Encounter.cpr_id = dbo_p_Patient.cpr_id) INNER JOINdbo_c_User ON dbo_p_Patient_Encounter.attending_doctor =dbo_c_User.user_id
WHERE encounter_date > begin and encounter_date < end and indirect_flag = 'D'
ORDER BY dbo_c_User.user_full_name;

When run from MS Access, this will prompt the user for a begin date and an end date.
For example, if you want to know who were the attending providers for only 9/21/2005, begin = 9/21/2005 and end = 9/22/2005
