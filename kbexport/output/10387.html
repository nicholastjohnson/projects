Re-Diangosed Assessments do not print out on Encounter Report

10387

Introduction

Symptom

"Current Encounter Report" and "Encounter Report" do not print Assessments that have been re-diagnosed

Cause

Workaround

Resolution

Version 2.0 (This must be done per PC and when EPRO is not in use)

  • Stop all services and end all EncounterPRO.exe processes in Task Manager
  • Rename epreport.pbd to epreport.old (program files\jmj\encounterpro)
  • Download latest epreport.pbd from FTP site \EP_Quick_Fixes\2.21\Enctrrptfix.zip
  • Unzip and copy file in program files\jmj\encounterpro
  • Start all EPRO Services

Version 4.0

Create issue and assign to Customization to Configure Report
