Symptom

Some sites want additional functionality in the Avg_Exam_Time rtf report, as the Crystal report does not work for them.

Until the new report verison is 'released' from 'testing' you can add to site avg exam time rtf by doing the following

On WWWServer ftp ep_reports folder thers is a sql stored procjmjrpt_Service.PRC. Open this up with notepad and copy the stored procto the sql query for the cpr database. Install the stored proc.

Then go into the Configuration/Display scripts/General

Find the Avg Exam Times rtf, make a local copy.

Edit the local copy.

Append a new general query

Put the following in the query window...

 exec jmjrpt_Service @ps_SERVICE='%service%',@ps_begin_date='%begin_date%',@ps_end_date='%end_date%

 

Set column headings to True

Set grid lines to True

Test your work
