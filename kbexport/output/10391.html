How to Release a New Build

10391

Introduction

To release a build to Production or some other status:
If this build is in a system that EP Release Manager cannot set toproduction (like Outlook forms), then go to database "issues" (on server Techserv), tabler_compile and set the build_status to Production and set the releasedate to today

Else, go to EP Release Manager, select the system, click Release, select the build you want to release, click OK, specify whether you want to release to Testing, Beta or Production, click OK, specify the Minimum Mod Level, OK, Maximum Mod Level, OK. This will release the sytem.

To create a new build:
Go to table "build" and insert a new row : enter the system_id, the major release, minor release and extended release.
