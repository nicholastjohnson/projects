Unable to add a new result to Physical Exam

10397

Introduction

Symptom

User is unable to create a new result

Cause

The happens when the sort_sequence field is maxed out for the SKIN observation. Fortunately the specific value is not important for the sort sequence field so we can shift all of the sort sequences down by 5000 using the following query.

First locate the Observation ID and replace SKIN

UPDATE c_Observation_Result

SET sort_sequence = sort_sequence - 5000

WHERE observation_id = 'SKIN'

GO
