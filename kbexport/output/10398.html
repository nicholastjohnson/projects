Nothing happens on clicking Display for attachment

10398

Introduction

Symptom

For some or all image attachments, nothing happens when you select 'Display'.

Error reported in event viewer:

System.OutOfMemoryException: Out of memory.
at System.Drawing.Image.FromFile(String filename, Boolean useEmbeddedColorManagement)
at EPImageControl.ImageControl.LoadImageFromFile(String fileName, Boolean readOnly)

Cause

In much older versions of EPImageViewer, before we started to use Atalasoft libraries to open images, this 'Out of memory' error was returned by the standard .NET imaging libraries when a particular image could not be read.  Usually it was a TIFF compression that was not supported.

Workaround

Resolution

Upgrade EproLibNET to the latest supported version.
