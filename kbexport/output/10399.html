How to Create & Check a Unix Mount

10399

Introduction

To Create a Unix Mount,

  1. Log into the practice's main/Windows server.
  2. Telnet into the practice's Unix server - the IP address and root password should be in the customer profile
  3. cd /etc/rc.d/8  (to go to the directory where some boot up scripts are stored)
  4. vi automount_JMJ (to create the file)
  5. i (to enter into Insert mode of the vi text editor)
  6. Enter the following lines (for example) after you have ready
    1. Internal IP of the Windows server
    2. NFS path of the Windows shared folder
    3. Unix path of the mounted folder
  7. echo Mounting JMJ share
    mount -f NFS 192.168.100.1:/jmj/pmshare /usr2/jmjshare
    echo JMJ share mounted
  8. :wq (To write (save) changes to disk and quit vi. Note the : before the w)
  9. chmod a+rx automount_JMJ (to make it readable and executable by all users)
To check to see if the pmshare directory is mounted:
From the command prompt of the Unix server, just enter the command "mount"

To verify the Unix server is able to write to the pmshare directory:
  1. Go to the jmjshare folder (Example: cd \usr2\jmjshare)
  2. Create a test file (Example: vi test.txt)
  3. Save changes to the file and close it - :wq
  4. Verify the file was created and changes saved in the pmshare folder in the Windows server
To Unmount a directory
    Use the 'umount' command with the name of the directory on Unix.
    Ex: umount /usr2/jmjshare
