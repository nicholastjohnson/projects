Displaying telephone call info on a report

10400

Symptom

Cause

Workaround

Resolution

Make sure you're in Config mode.  Go to the report that needs modifying - usually by clicking on the sigma button on the SOAP screen. Right click on the white part of the screen.  Make sure you've got the Local version highlighted, or make a local version if necessary.  Click on Edit Display Script.

Editing the Encounter SOAP Note Standard display Script:

  • Scroll down, clicking on each command on the left that says "encounter treatment" and looking at the box on the right until you see one that says treatment_type = Physical
  • Insert New Command - Encounter Context, Treatments
  •      Treatment Type - Telephone Record
  •      Show Previous - False
  •       Show New Treatments - leave blank
  •       Include Created in Plan - leave blank
  •      Treatment Status - leave blank
  •      Encounter Owner - leave blank
  •      Display Script - click on the box, click Display Name, and type in Phone and click OK.  Select Telephone History Display and OK.   Click Edit below the display script box to edit it.

Editing the Telephone History Display -

  • Disable the Treatment Query command
  • Insert New Command - Treatment Context, Services
  •      Service - Epro Todo Item
  •      Which Services - All
  •      Format - Message
  •      Empty Phrase - Blank

Back to the larger display command that we were in -

  • Empty Phrase - leave blank
  • Non-Empty Header Phrase - Telephone Histories:
  • the rest of the attributes can be left blank.
