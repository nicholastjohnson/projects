File not found on scan post screen

10403

Introduction

 

Symptom

User is unable to see item on to be posted screen. An example of an error is "Error 53 File Not Found"

Cause

This happens when a workstation does not have correct rights to the temp folder and CPRFILES.

Workaround

Resolution

Follow the below steps

  • Ensure that all users have read and write access to c:\temp on database server and application server
  • Ensure that CPRFILES have read and write for all
  • If workstation is a fat client ensure that they have a c:\temp
  • Ensure that a weekly job is set up on the scanning PC and the server to clear all JMJ* files from the temp folder
