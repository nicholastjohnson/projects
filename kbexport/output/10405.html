Moving Data and Attachments to a separate Drive

10405

Introduction

This provides instructions on how move the attachments or all of the EPRO data files from one drive to another

If the attachments are not already seperated from the cpr data. Follow below KB article

https://www.jmjtech.com/kb/Article.aspx?ArticleNumber=10332

Symptom

Cause

Workaround

Resolution

Follow the below steps. All Users must be out of EncounterPRO

  • Open up Enterprise Manager
  • Right click on EncounterPRO Database (Usually cpr)
  • Select All Tasks and Detach Database (If any connections are active select the Clear Button)
  • Copy attachment file group over to new drive
  • Right click Databases in Enterprise Manager
  • Select All Tasks and Attach Database
  • Select "..." and locate files
  • Change Data drive for attachment location
  • Open EPRO Server and if Login is not set select "Set Login"
  • If Security Set is set to "Standard" select "Set Application Role"
  • Confirm that scanned documents open and that you can chart and encounter
  • Once practice confirms that the data has copied you can remove the data
  • Set up SQLJobs on new Drives
