Well visits billing as Sick visits

10409

Introduction

This article explains when well visits would be billed as sick visits

Symptom

Cause

a) If it's a WELL visit but they ordered only SICK diagnosis, not WELL, then it bills them as SICK visits.

b) If it's a medicaid patient and if they ordered both SICK and WELL diagnosis, then there is an attribute that controls whether to bill a Normal code or abnormal code (see medicaid billing rules KB article to know more)
