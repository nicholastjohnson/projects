Ricoh Printer Settings

10410

Introduction

In order for the prescriptions to properly work you have to have the printers added as two seperate printers

 

 

Symptom

Cause

Workaround

Resolution

If a virtual printer is not created you will need to create one or have the hardware team create one

Right-Click the Printer and select Properties

Select the General Tab

Click the Printing Preferences Button

Select the Paper Tab

Change the “Input Tray” to “Bypass Tray”

Click the “Custom Paper Size” Button

Change the Unit of Measure to Inch

Change Width to 4.25

Change Length to 5.5

Navigate back to Printer Properties

Select the Advanced Tab

Click the Printing Defaults Button

Select the Paper Tab

Change the “Input Tray” to “Bypass Tray”

Click the “Custom Paper Size” Button

Change the Unit of Measure to Inch

Change Width to 4.25

Change Length to 5.5

Navigate back to Printer Properties
