Setting up a LabCorp customer on EpIE

10413

Introduction

This article describes the steps for setting up a Labcorp customer to automatically get lab results from Labcorp and process them through EpIE and download them to the customer database.

A number of Labcorp Customers were deployed using the BIE 5.0 system.  This system has been superceded by the EpIE (EncounterPRO Integration Engine) system.  For these customers some of the steps will have already been performed.

Cause


Workaround


Resolution

The steps for setting up a Labcorp customer on EpIE are as follows:

 

  1. Create the Addressee record (if it doesn’t exist yet).  Set send_flag to ‘N’.
  2. Create a customer-specific EpIE Messages directory on Gateway and make sure it is accessible from EIE
  3. Configure the Receiver component to pull messages from the customer-specific EpIE Messages directory on Gateway
  4. Contact Labcorp for customer specific information for Dialer Script (port # etc)
  5. Create the Dialer Script on Gateway to pull messages from Labcorp and place them in the customer-specific EpIE Messages directory
  6. Create credentials on WWWSERVER:  user = customer_id, pwd=
  7. Test messages by manually placing a Labcorp test message into the customer-specific EpIE Messages directory on Gateway.
  8. When Labcorp is ready to send test messages, test Dialer Script.
  9. Make sure “JMJ Lab Results” com_id attribute is set to Eprolibnet.IntegrationEpIE
  10. Make sure the owner_id field in c_Consultant is set for the lab's c_Consultant record
  11. Configure External Source service in o_Service_Schedule and o_Service_Schedule_Attribute
  12. Test download by changing a few test messages from ‘Ready Hold’ to ‘Ready’
  13. When ready to turn on customer, set Dialer Script to run at scheduled time and set m_Addressee.send_flag to ‘Y’ for customer’s AddresseeID

Step 9.  To configure a customer to use EpIE instead of BIE 5.0, change the com_id attribute of the JMJ Lab Results external source using the following query:

UPDATE c_External_Source_Attribute
SET value = 'Eprolibnet.IntegrationEpIE'
WHERE external_source = 'JMJ Lab Results'
AND attribute = 'com_id'
GO
