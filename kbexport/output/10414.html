Requirements:

PC must be connected to the network

 

Step1: Installing the Citrix client

 

The ICA32 client programshould be on your server under support, Citrix, ICA client 32.

ie. : \\support\citrix\ica client32

 

Copy this to you remotecomputer or you can download it from the following web site:

http://www.citrix.com/download

 

Once you have copied ordownloaded this client on your remote PC you need to install it.

 

Double click on the ICA32Icon

Click Next, then Next, thenYes, then Next, then Next,

Enter your computer name oruse the default. Then click Next, and Next again.

The client then install onyour remote PC and when it is finished click on OK

You have now installed theClient on your remote PC. 

 

Step2: Setting up the Citrix client

 

Directions to connect to the server via the LAN.

 

Goto Start , Programs, Citrix ICAClient, Citrix Program Neighborhood

TheCitrix Program Neighborhood should open up.

DoubleClick on Application Set Manager

DoubleClick on Custom ICA Connections

DoubleClick on Add an ICA Connection

Select“Local Area Network” and click Next

“Entera description on the new ICAconnection…”

= Encounter PRO

“Selecta Citrix server or published application to which you want to connect…”

                = (You server’s internal IPAddress)

Clicknext, then Next

Enteryou User Name , Password, and Domain name, then click next.

Uncheck“Use default” for Windows colors and select “Hi Color (16Bit)” or

  “True Color (24Bit).

Thenclick Next,

Enterthe file name of the application to run…..

  D:\ProgramFiles\jmj\encounterpro\encounterpro.exe client

Thenclick Next.

ClickFinish and you are done. 

To access you server justdouble click on the Icon you have created in the Citrix Program Neighborhood.You can drag it onto your desk top as well
