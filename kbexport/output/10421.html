Epro prints a blank page whenever they attempt to print

10421

Introduction

Epro prints a blank page whenever they attempt to print particular reports like

"General VFC Usage" report.

Vaccine Report (Immunization Record-Maker/Lot)

Symptom

Produces a blank page

Cause

Old version of the report, or permission to run report was revoked.

Workaround

UntilReport version7 is 'released' from testing

do the following

at site bring up \\Wwwserver\ftproot\EP_Reports

for General VFC Usage

open the files:

jmjrpt_vfc_rpt1.sql

jmjrpt_vfc_rpt2.sql

Open up SQL Query Anaylzer, and set dbatbase to the Encounterpro production db.

Past in the query and run

For Immunization Record-Maker/Lot

open the file jmjrpt_patient_immunization.PRC

past into a sql query window and run.

Test the reports.
