Resolution

Taking out a top tab (example: H&P on the SOAP screen)

  • Go to c_chart_section_page
  • filter for the desired tab in the description field
  • for all of the rows returned, add 4 zeroes to the end of the chart_id (so the record won't be read)

 

Fixing the Flow tab:

  • Go to c_chart_section_page_attribute
  • in Value, filter for   *general*general*
  • will return rows with the value  %General %General Favorite Quick View, PT Flow%
  • delete the first %General so that it just has    %General Favorite Quick View, PT Flow%
  • the value will now look for the preferred PT Flow script
