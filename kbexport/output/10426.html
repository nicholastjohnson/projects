Introduction

This article explains what must be done to install Primetime's Instant Medical History (IMH) so it can integrate with EncounterPRO.

IMH Instant Medical History allows a patient to enter detailed medical history data and to have it appear within EncounterPRO automatically.

Cause


Workaround


Resolution

Currently, the steps for installing and testing IMH are

1.      Install and configure IMH
2.      Install and configure production build EncounterPRO Client
3.      Test integration

 

1.Install and configure IMH

Download IMHG install files from http://www.medicalhistory.com/download.htm
Username = imhuser
Password = getmyimh

  (You’ll find the IMH.exe in C:\Program Files\Primetime Medical Software\Primetime Instant Medical History)

When it asks for your license name and number, use

Dr. Chuck Webster
8804-4955-7738-0126

Run IMH, cancel questionnaire, go to Tools -> Options… -> EMR (Tab)
Check Output EMR report…

Under EMR type, Select “EncounterPRO”


Click EMR Report Output Location

Set it to C:\temp (create folder if necessary)

Select File Name Options…

Check Use patient ID for file name

And check off Overwrite the existing file name

(These steps told IMH to output an XML file that can be automatically imported into EncounterPRO. We do not currently import this file but may do so in the future. We are basically preparing for that eventuality. The next set of operations tell IMH to generate a wri file in the rtf form. This is the file that EncounterPRO currently imports and displays.)

Select Report Preview Tab

Check Auto-create report file. Click on Report Preview Output Locations



Set Primary output location to C:\temp



Select Report File Type and set to Rich Text Format


Select Filename Options and set the to the same as above

3. Test integration
