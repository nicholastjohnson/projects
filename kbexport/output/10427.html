how to send referrals automatically to referral user version 2

10427

Introduction

this article explains how to automatically send the referral to-do items to referral user.

Symptom

referral user is not automatically assigned in the referral screen. ie. send todo item is assigned with value of "None"

 

Cause

a) the auto referral is not enabled

b) the referral user is not assigned

 

Workaround

Resolution

make sure the following attributes are defined in o_preferences

preference_type

user_id

preference_id

preference_value

REFERRAL

!JMJ

todo_user

0001REFERRALS

preference_type

user_id

preference_id

preference_value

PREFERENCES

!JMJ

auto_referral_todo_item

YES

 

Note:  If the referral user not showing up then make sure they exit encounterpro and re-login. Because we cache the "preferences" table
