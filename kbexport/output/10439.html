Error creating local copy of display script

10439

Introduction

After a config database is installed at a new site, the user is unable to create a local copy of a display script.  This article explains how to repair the customer database so that local copies can be created.

Symptom

The user selects "Create local copy" within the display script configuration screen.  Nothing happens.

The error message in the event viewer is

Cannot insert the value NULL into column 'script_type', table 'cpr_405_beta.dbo.c_Display_Script'; column does not allow nulls. INSERT fails.

Cause

Workaround

Using DB Maintenance:

1) Make sure Hotfix 92 has been run already

2) Run Hotfix 94

 

Resolution

A dev issue has been logged for a permanent fix.
