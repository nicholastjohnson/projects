The Chart for a specific encounter type is wrong

10440

Introduction

Sometimes a telephone encounter or other type looks a little funny when you view chart during the encounter.

Symptom

Cause

The wrong Chart is being used for that encounter.

Workaround

Resolution

C_encounter_type to find out the encounter type

C_workplan_selection - search for the encoutner type to find out the workplan_id being used

To find out which chart the practice usually uses (for peds, IM, and FP no OB)

U_chart_selection, filter for the practice's specialty.  The default is the chart_id in the record that doesn't have a workplan_id

U_chart_selection - filter for the workplan_id you found earlier and change the chart_id to the default.
