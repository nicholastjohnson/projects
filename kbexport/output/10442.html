Cannot get past Logon Screen to EncounterPRO

10442

Introduction

Symptom

Below is the error that you get when you open EncounterPRO. Sometimes it will take you to the logon screen. In all known cases it will not let you log into EncounterPRO

u_sqlca - (check()) SQL ERROR = (8114) SQLSTATE = 22018

Microsoft OLE DB Provider for SQL Server

Error converting data type varchar to int.

Cause

This happens because MDAC is not properly working on the PC

Workaround

Resolution

  • Uninstall all EncounterPRO components (Do not unistall EPRO if you do not see a seperate Change and Remove icon and if you do not see a support information tab that you can click on)
  • Install MDAC 2.8  http://www.microsoft.com/downloads/details.aspx?FamilyID=6c050fe3-c795-4b7d-b037-185d0506396c&DisplayLang=en
  • Install MDAC 2.8 SP 1 http://www.microsoft.com/downloads/details.aspx?FamilyID=78cac895-efc2-4f8e-a9e0-3a1afbd5922e&DisplayLang=en
  • Run Comchecker http://www.microsoft.com/downloads/details.aspx?FamilyID=8f0a8df6-4a21-4b43-bf53-14332ef092c9&DisplayLang=en
  • If MDAC will not install advise client to contact IT company
  • Comchecker needs to return little or no errors and conflict files
  • Install EncounterPRO
