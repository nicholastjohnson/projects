Bills not getting picked up by PM Integration

10456

Introduction

This article explains what to do if bills are confirmed to betransferring from EncounterPRO to the correct folder, but the files arenot picked up by the practice management system.

Symptom

1- Confirm that the folder where bills are being sent is the folder that the PMS is looking at
2- Confirm the PMS has read/write access to the folder so it has permission to delete the files after they are read.
3- If this is an integration that involves a Unix server (like Medical Manager), verify that that the mount is correctly setup.
4- If the integration is Medical Manager and it is giving an errorabout no available filenames, goto the pmshare folder and delete thebilling files
