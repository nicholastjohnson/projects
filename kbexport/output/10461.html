Mapping test results to allergens

10461

Introduction

The allergy vial definition screen allows the user to establish mappings between allergy tests and specific allergens.  The user can create or remove mappings one at a time, but if the user wants to start from scratch and remap all tests, there is no easy way to do that.

Symptom

Cause

Workaround

The mappings can be removed from the EncounterPRO database with the following query:

 

DELETE x
FROM c_XML_Code
WHERE code_domain = 'allergen_observation_id'
AND epro_domain = 'allergen_drug_id'
GO
