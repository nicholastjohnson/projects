PFSH child observation not appearing

10468

Introduction

Symptom

A simple child observation in a PFSH on a Yes/No screen is seen on the observation configuration screen, but not on the Enter Results Yes/No screen.

Cause

The observation has a location domain attached to it.

Workaround

Resolution

Remove the location domain and add a comment result, if possible, and add the locations to the "specific" pick-list.
