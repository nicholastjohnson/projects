Lab Integration - Extraneous Lab discrepancy

 

Note: In some instances, Encounter Pro is attempting to generate an extraneous generic lab after posting lab results to a treatment. You will notice on the screen capture below that the treatment type is “Lab” and the description field has the word “Lab”. When this situation occurs make sure to cancel back to the Scan/Post utility.

 

 

If you select new treatment, Encounter Pro will generate a generic lab. If this happens, do not post any lab results to the generic lab. You can close the lab from the Outstanding Labs/Test Utility and/or cancel the lab from the SOAP screen.
