Workaround

Resolution

Go to Configuration, Workplans, change the workplan type to Treatment, and click New Workplan, and select Lab.  Workplan description can be something like "Lab, Send Out to " First Step: Set Perform Mode    Edit Step, Add Service, New Note, Context Object = Treatment, Special Note Type = Property,     Key = Specimen Sent To:, Note Text = [enter description from c_consultant] (CASE SENSITIVE)       Edit description = Set Perform Mode for %WP%      Ordered For = Roles: Any Clinical Staff       Edit Flags = In Office, Counts toward Step Completion  = Y, Perform Immediately = Y, the rest are No or NA Second Step: Set Consultant_id
    Edit Step, Add Service, New Note, Context Object = Treatment, Special Note Type = Property,
    Key = consultant_id, Note Text =
       Edit description = Set Perform Mode for %WP%
       Ordered For = Roles: Any Clinical Staff
       Edit Flags = In Office, Counts toward Step Completion  = Y, Perform Immediately = Y, the rest are No or             NA
Third Step: Collect
    Edit Step, Add Service, Specimen Collection,
       Edit description = "Collect %WP%"
       Ordered For = Roles: Any Clinical Staff
       Edit Flags = In Office, Counts toward Step Completion  = Y, the rest are No or NA
       Click OK
