Changing display from abnormal only to all

10475

Introduction

The doctors have notcied a change in E-Pro after the update that they'd like to know if can be changed back to the way it was prior.  When they go into a patient's chart under quick view history & physcial it only shows abnormal things under ROS for adults and general physical.  They would like it to be returned to the way it was so it will show all results, normal & abnormal. 

Symptom

Cause

Workaround

Resolution

Go to Configuration and Display scripts.  Change the context to Treatment.

For each of these display scripts

Standard Quick View ROS Display
Standard Quick View ROS Yes/No Display
Standard Quick View Physical Display

Edit the script -

    Take out any print command that prints the string " (Abnormals) "

    Edit the Treatment Results command
       Abnormals Only = No
       Edit Empty phrase to remove "abnormal"
