10476

Introduction

this article explains how to fix cic signature issue for migrated clients. actually the cic signatures are not migrated during the initial migration phase.

Symptom

(cic) signatures are not showing up in Rx and Immunization reports after migrating from version 2 version 4.

Cause

(cic) signatures are not migrated during the initial migrations.

Workaround

there is a fix included in 390 end migration.sql. this was fixed on 3/20/2006. so any practices migrated earlier to this date might experience the signature copying issues. so re run the 390 End Migration.sql script to fix them.
