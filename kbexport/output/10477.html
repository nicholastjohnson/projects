Unable to create SQL Jobs

10477

Introduction

Symptom

Below is the error that appears when you try to create the SQL Jobs

Server: Msg 14262, Level 16, State 1, Procedure sp_add_jobserver, Line 82
The specified @server_name ('SERVERNAME') does not exist.

Cause

This happens because the server does not recongize the @@SERVERNAME

Workaround

Resolution

FIND and REPLACE @@SERVERNAME with 'SERVERNAME' (include the quotes)

Notes
