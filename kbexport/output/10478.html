Adding buttons to the treatment dashboard

Introduction

Symptom

Cause

Workaround

Resolution

Go to Configuration and Treatment Types
Click on the desired treatment type, then Edit
Pick the Services tab at the bottom of the screen
Click Add Service
Select the appropriate Context on the right, and then the service on the left.
    (example: Edit Lab/Test Drilldown)
Find the service that you just added on the left and click on it, and Edit Service
    Set the "Visible before workplan Completes" to Yes
    Rename the Button Title to whatever you want the button called.
    Click OK
Click on the service again and choose Configure Service.  This takes you to the same set of screens that represent the possible service attributes that you would get if you added the service to a regular menu.
Click OK and Finished.
