Creating a General Report

10479

Introduction

Symptom

Cause

Workaround

Resolution

1) Create the report in c_report_definition
    Report type - general, component_id - RPT_RTF
2) Create a Display Script (or find out which display script you're using)
    c_display_script (info also in c_display_script_command and c_display_script_cmd_attribute)

There needs to be a display script command "Query" with a value something like:
exec jmjrpt_treatment_bydate
@ps_begin_date = '%begin_date%',@ps_end_date = '%end_date%',
@ps_observation_id = '%observation_id%'

3) Set Attributes
    Go to c_report_attribute and create a row with the report_id of the report, attribute of display_script_id and value of the display_script_id of the display script that you want to use

4) Set Parameters
    Go to c_component_param_class and pick which ones you need for the stored procedure that you're going to use.  Add that to c_component_param with a param mode of Runtime.  The Tokens are what get passed to the stored procedure, so just use whatever you use there.

5) Create Stored Procedure
(At the bottom, add the line "grant execute on jmjrpt_whatever to cprsystem")
