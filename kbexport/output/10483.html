EncounterPRO installs only for current user and will not open for other users

10483

Introduction

Symptom

EncounterPRO icon does not appear on other users' desktops and will not work when run manually. EncounterPRO works when opened while logged in as user that performed the installation.

Cause

ALLUSERS setting appears to be overridden by Windows on some machines. This will cause EncounterPRO to be installed only for the current user.

Workaround

Install EncounterPRO by running setup.exe from the command prompt with a parameter to set ALLUSERS.

Example:
C:\>"D:\Install\JMJ\Epro Client Setup\setup.exe" /vALLUSERS=1
