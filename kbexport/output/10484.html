Print supervisor's sig on report

10484

Introduction

On a report, we only want the encounter owner's supervisor's information to print if the encounter owner actually has a supervisor...

Symptom

Cause

Workaround

Resolution


Encounter IF  True/Talse query under the encounter owner's print commands:

SELECT 1
FROM c_user u
INNER JOIN p_patient_encounter pe WITH (NOLOCK)
ON u.user_id = pe.attending_doctor

WHERE pe.encounter_id = %encounter_id%
AND pe.cpr_id = %cpr_id%
AND u.supervisor_user_id IS NOT NULL

True display script can be something like "Encounter Supervising physician signature and name"
