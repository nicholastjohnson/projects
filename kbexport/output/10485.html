Retry Posting in v2

10485

Introduction

This article explains how to resend billing in EncounterPRO version 2

Symptom

Cause

Workaround

Resolution

  1. Stop the Billing and TranOut service.
  2. Run this query from Query Analyzer:
    Update p_Patient_Encounter set billing_posted='N'

    where encounter_date >  [enter the date in single quotes] and bill_flag = 'Y' and billing_posted = 'Y'


     3.   Start the Billing and TranOut service

Notes

In order to retry the posting of just one patient, use steps 1 and 3 above, but instead of step 2:

  • Go to p_patient, get the pt's cpr_id based on their billing_id.
  • Next go to p_patient_encounter and filter for this pt's cpr_id and find this particular encounter. (Clues : the indirect_flag should be set to D and bill_flag will be Y.)
  • Next go to the billing_posted column for this patient's encounter and set it from Y to N.
