Config Upgrade Error Cannot insert duplicate key in object 'c_Drug_Definition'

10490

Introduction

Symptom

While performing a config upgrade, the following error message is reported:

SQL ERROR = (2627) SQLSTATE = 23000
Microsoft OLE DB Provider for SQL Server
Violation of PRIMARY KEY constraint 'PK_c_Drug_Definition_1__10'. Cannot insert duplicate key in object 'c_Drug_Definition'.


 

 

Cause

This error is caused by a mismatch between records in c_Vaccine and records in c_Drug_Definition.

Workaround

This error is repaired by Hotfix 98.

1) Run Hotfix 98
2) Restart Config Upgrade

Resolution

This error is permanently fixed in Mod Level 127.
