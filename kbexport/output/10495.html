Get a list of patients diagnosed by a given diagnosis Crystal Reports error calling function printreport

10495

Introduction

This article explains how to retrieve the list of patients diagnosed by a given diagnosis. Also explains how to troubleshoot some of the generic report errors and Crystal reports.

Symptom

Most Common Error:

u_component_service_report - (do_service()) Error doing service (Error calling external object function printreport at line 35 in function xx_printreport of object u_component_report.)

Mostly the above error occurs if EncounterPRO unable to execute the report which means the report is not properly configured or installed in the case of Crystal reports.

Cause

Workaround

Resolution

Installing EproCrystal85Runtime.exe from ftp://ftp.jmjtech.com/EP_SupportingInstallers may fix this problem.

Notes

Version 4. How to fix patients with Dx Crystal Report:?

Find the report id in c_report_definition for that report and check to see the "REPORT FILENAME" attribute is set with a valid path that contains the Crystal report files. Usually all the Crystal reports are installed under CPRFILES\Reports. If you do not see a reports folder that means the Crystal reports are not installed.
