Office drugs have packages, packages have administration methods, and administration methods have procedures associated with them.  In the tables, this can be seen as

c_drug_definition.drug_id = c_drug_package.drug.id  ->  package_id
c_drug_package.package.id = c_package.package_id -> administer_method
c_package.administer_method = c_administration_method_proc.administer.method -> Procedure_id
c_administration_method_proc.procedure_id = c_procedure.procedure_id -> description
