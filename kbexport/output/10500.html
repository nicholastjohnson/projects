Patients by same name cannot be checked into same exam room

10500

Introduction

this article explains how to resolve patients by same name and not able checked into same exam room

Symptom

two patients by the same name and one of them vanished while checking into exam room. but showing two service bars to take vitals

Cause

It's because EncoutnerPRO supress the duplciate bar names on the left side. If two of the patients you are checking in has the same first name and last name then it'll supress one patient bar.

Workaround

Build 35 address this little differently. it includes the name and age on the patient bar and so this should be resolved with 35. In case if the patients are twins and has the same first name and last name then it might be the same problem again. In that case we might include Nick names in the service bars to avoid those situations in future builds. I hope the fix implemented in 35 (name and age) should be fine now.
