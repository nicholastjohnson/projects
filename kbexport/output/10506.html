GRITS Config error

10506

Introduction

This article explains what to do if you get this error while configuring a GRITS report.

Symptom

From KB 10191, in step 4, nothing happens when you select the report to configure and you get this error in Event Viewer:

u_reports_edit - (u_report_edit) Error getting report component (GRITS_REPORT)

Cause

1- The file eprolib.dll is not registered.
2- The wrong eprolib.dll is installed
3- missing row in c_component_registry

Workaround

1- Re-register this file here:
D:\Program Files\JMJ\EncounterPRO

2- reinstall the production build of EproLib4 and EproLibNet

3- see KB 10298
