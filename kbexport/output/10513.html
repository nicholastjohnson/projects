Hotfix 119 - Last Well Visit

10513

Introduction

Per Support issue 22598, this hotfix enables the Last_Well_Visit report tag.

Symptom

Practice wants a report that includes the date of a patient's last well visit.

Cause

Workaround

Resolution

Run Hotfix 119. Update dev issue 6880 if you find any problems.

Notes

The "last well visit" is the last encounter in which there is an assessment (in p_assessment) where the assessment_type='WELL'
