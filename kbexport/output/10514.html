Growth Charts not working

10514

Introduction


Symptom

The growth charts do not display and an error like this one is reported to the event viewer:

Error accessing external object property datainit at line 12 in function init_graph of object u_cpr_page_growth_percentiles

 

Cause

This error is caused by the Flipper Graph Control not being registered properly.

Workaround


Resolution

1) Reinstall EncounterPRO

2) If that doesn't work, try manually registering the Flipper Graph control (FlpGrf.ocx) AND the Flipper Graph dll (FlpGrfADO.dll) using regsvr32.exe.  It will be either in windows\system32 or in the program directory.  If this is on a terminal server. make sure you run "CHANGE USER /INSTALL" before registering and "CHANGE USER /EXECUTE" afterwards.

 3) If that doesn't work, try searching the entire drive(s) for FlpGrf.ocx AND FlpGrfADO.dll.  Delete all copies found, then rerun the EncounterPRO Installation and choose the "Repair" option.  If this is on a terminal server. make sure you run "CHANGE USER /INSTALL" before repairing and "CHANGE USER /EXECUTE" afterwards OR run the installation from Add/Remove programs.
