Hotfix 120 - Last Well Visit Doctor

10516

Introduction

Per Support issue 22598, this hotfix enables the Last_Well_Visit_Doctor report tag.

Symptom

Practice wants a report that includes the full name of the attending doctor of a patient's last well visit.

Cause

Workaround

Resolution

Run Hotfix 120. Update dev issue 6896 if you find any problems.

Notes

The "last well visit" is the last encounter in which there is an assessment (in p_assessment) where the assessment_type='WELL'
