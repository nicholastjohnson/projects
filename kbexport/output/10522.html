EncounterPRO Pediatric Growth Charts

10522

Introduction

This article describes how EncounterPRO's Pediatric Growth Charts are configured and updated.

Symptom

Cause

Workaround

Resolution

EncounterPRO's Pediatric Growth charts are derived using the CDC's data and methodology  which are set forth at http://www.cdc.gov/nchs/about/major/nhanes/growthcharts/datafiles.htm

The CDC data are divided into 8 categories - each of which is represented by a separate cdc table in EncounterPRO's database.  When the CDC changes its data sets, these changes will be promulgated in EncounterPRO's cdc tables when the customer's database is synchronized or upgraded. 

The CDC's methodology for calculating growth chart values is hard-coded in EncounterPRO.  Any changes in the CDC's methodology would be promulgated to the customer in software upgrades.
