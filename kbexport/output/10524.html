How to Create a HIPAA field at the top of the SOAP screen

10524

Introduction

This article describes how to add a HIPAA field at the top of the SOAP screen in Version 4.0 or higher. 

Symptom

Cause

Workaround

Resolution

In the c_property table, there is a user-defined property (property_id 281) called "HIPAA Acknowledgement Date."  To link this property to a field at the top of the SOAP screen, go to the o_service_attribute table and follow the instructions below for each of three services:  "CHART," "EXAM," and "SUPERVISOR." 

Enter an attribute called "property_title_x" where "x" is an integer between 1 and 7.  The value for this attribute should be "HIPAA." 

Enter an attribute called "property_x" where "x" has the same value as was selected in the instruction above.  The value for this attribute should be "Patient HIPAA Acknowledgement Date."

Refresh EncounterPRO.  The HIPAA field should be added in the place (1 through 7) that was selected above for "x."
