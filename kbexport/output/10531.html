Hotfix 123 - c_preference Medicaid_Supress_Supervisor_Role

10531

Introduction

Per Support issue 25963, this hotfix enables the Medicaid_Supress_Supervisor_Role preference to alter the Medicaid billing rules.

Symptom

Practice wants a all users of a certain role to bill differently for Medicaid patient encounters- differently in that the users of this role will bill as themselves, not as their supervisor.

Cause

Workaround

Resolution

Run Hotfix 123.
After you run the hotfix, goto

  • Configuration
  • Preferences
  • Preference Type = Billing
  • click on "Suppress the supervisor being the billable provider for Medicaid"
  • set the role(s) you want this to apply to (Usually NP).

Update dev issue 6935 if you find any problems.
