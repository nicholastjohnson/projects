Policies and Timetables for Updating CPT and ICD9 Codes

10536

Introduction

This article describes JMJ's Policies and Timetables for Updating CPT and ICD9 Codes.

Symptom

Cause

Workaround

Resolution

It is JMJ's policy to provide annual updates of CPT and ICD9 codes to user sites on a timely basis.  It is JMJ's policy to create sequel scripts prior to the effective dates of these code updates.  After a site grants written permission to run these scripts, JMJ deploys them as protected transactions.

User permission is required to run JMJ update sequel queries because EncounterPRO's user interface provides the ability for users to update their own codes.
