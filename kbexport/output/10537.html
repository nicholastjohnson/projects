Compatibility Issues with DEP Data Execution Prevention

10537

Introduction

This article discusses an EncounterPRO compatibility issues in regards to DEP (Data Execution Prevention) default settings when running on NX compatible hardware.

Symptom

EncounterPRO has compatibility issues in regards to DEP (Data Execution Prevention) default settings when running on NX compatible hardware.

Cause

Workaround

Resolution

Detailed information about DEP is available from Microsoft KB article 875352. One Micorsoft-approved workaround is to change the Default Setting from "Turn on DEP for all programs and services except for those I select" to "Turn on DEP for Esssential Windows Programs and Services only". We believe that this provides sufficient security for critical opperation of the systems. This is the only off-the-shelf malware detection and eradication software that has created problems.
