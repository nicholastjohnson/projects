Compliance with NTP SNTP Synchronization Standards

10538

Introduction

This article describes how EncounterPRO complies with NTP/SNTP time synchronization standards and describes how such timing is used in all security records maintained by the system.

Symptom

Cause

Workaround

Resolution

The Microsoft SQL Server environment provides all records of time for the EncounterPRO application via the GETDATE() T-SQL function.  This function returns the time of the underlying Microsoft Windows operating system.  Microsoft has included the Windows Time Service in its server operating systems since Windows 2000.  The Windows Time Service coordinates the time for all computers in a domain and can synchronize with an internet authority, as described in the article here: http://www.microsoft.com/technet/prodtechnol/windowsserver2003/technologies/security/ws03mngd/26_s3wts.mspx
