Protection of Private Information Against False Remote Nodes

10540

Introduction

This article describes how, when communications are necessary over open networks, EncounterPRO protects any PHI communicated against false remote nodes.

Symptom

Cause

Workaround

Resolution

The system uses user/password authentication for remote nodes when communicating PHI over the Internet using either SSL or IPSec VPN.  In the case of SSL, the host end point is validated by an X.509 Certificate.
