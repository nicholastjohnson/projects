Notes

As you know we do not have a good document that explains all the reports and it's output view. so whenever you are puzzled on a report is available in epro i recommend you try doing search against c_report_definition  / c_display_script table to see if you can find one or more and try to review them whether they are closer or matches to the customers requirements. if you are unable to find out the report you are looking for then make sure you follow this kb article https://www.jmjtech.com/kb/Article.aspx?ArticleNumber=10238  to get details on either customization existing or creating new reports.

For example, if a customer is looking for "anticipatory guidance" report then do below steps.

There are a few reports like this in the encounter and treatment context. I did a search against  c_display_script for 'Anticipatory Guidance' and tired find  a report that is linked to this report by doing a search in c_report_definition_attribute for that display_script id.

OR do a direct search in c_report_definition itself.
