Lab results don't show up on RTF report

10548

Introduction

Symptom

Lab results posted from an outside lab (e.g. Labcorp) appear on the lab treatment dashboard and on the Review Labs screen, but do not show up in a dated grid on an RTF report such as the OB Flowsheet.

Cause

When lab results are posted into the EncounterPRO database, EncounterPRO will automatically create observation and result definitions for any lab results that it doesn't recognize.  In some cases, the result definitions are getting an incorrect value for the [result_amount_flag] field.  in other cases, the lab company is sending an empty string as the result value and EncounterPRO is interpreting this as the latest result.

Workaround

This issue is resolved in Mod Level 127.  In mod level 126, this issue may be resolved by applying Hotfix 126.

Resolution

This issue is resolved in Mod Level 127.
