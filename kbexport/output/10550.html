How to separate the CC HPI ROS in to CC with HPI and separate ROS

10550

Introduction

We have noticed that the latest config of encounter pro (34.21) comes out of the box with the CC/HPI/ROS all on the same page. This complicated training. Below are instructions on how to configure the CC/HPI to be separate from the ROS… more similar to the training paradigm as put forth in our instruction manuals.

 

  1. Log on to Epro

 

  1. Select the Configuration Tab

 

  1. Select Preferences

 

  1. Set Preference type to “User Defined” and select “Speciality” at the bottom

 

  1. Change the Standard Favorite Child CC/HPI to “child cc with HPI” (may be set as inactive)

 

  1. Change the Standard Favorite Child ROS to “Remarkable Add’l History (old)”

 

  1. Change the Standard Favorite Child ROS treatment mode (button) to “Yes/No Format”

 

  1. Change the 120s to “Yes/No Format Provider”
