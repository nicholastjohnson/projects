No start items ordered

10551

Introduction

This article explains this error.

Symptom

Getting this error in the Event Viewer:

u_str_encounter - (add_start_items()) No Start Items Ordered.

Cause

Workaround

Find the cpr_id and encounter_id for the encounter that got this error.  

Look up the office_id in p_Patient_Encounter and the workplan_id in o_Patients.

Look in o_Workplan_Start to see what start items are configured for that workplan and that office. If there is no row for this office in o_Workplan_Start, add one.
