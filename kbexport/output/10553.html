Insurance not crossing from PM system

10553

Introduction

This article explains how to troubleshoot an issue where patient insurance information is not crossing from a practice management system.

1- Get more info:
Look up the PM system in the PM Integration Handoff document. See if it is HL7. If not AND the system is Lytec or Medisoft, then insurance info should not be crossing over. That is just how these systems work. Else, insurance info should cross over.
2- Get a patient example and have the practice try to change this patient's insurance info. Capture the file sent over from the PM system. If them PM system does not send a file, it may be the fault of the PM system.
