Missing patient attachments

10556

Introduction

This article explains the purpose of, and how to get and use the Missing Patient Attachments utility.

Purpose

This program will list all results of a query of patients whose attachments may be misplaced or lost. It will then show the number of patients whose attachments were misplaced but are now fixed. It will also show the patients whose attachments could not be found.

How to Get

This program can be found here:
ftp://ftp.jmjtech.com/JMJSupport/SupportTools/mpa.zip

Unzip the files onto the local computer's hard drive.  If you unzip the files into an EncounterPRO program directory, you may use a parameter to specify the EncounterPRO database (see below).

How to Use

Run this program by double clicking on it or running it from a command line.  If you run it from a command line, you may supply a parameter as follows:

mpa.exe [label]

Where label is the name of the database link shown on the ASK screen. For example, in this case:

the label would be Beta-Dellserv, not cpr_405_beta.
So in this case the program might be run like so:
mpa.exe Markdev

When the interface comes up, click “Verify/Repair Files”.

If the program is run with a valid paramter and in a folder where EncounterPRO.ini exists, it will use the label in that file. Otherwise, if no label is given or the label is invalid, you will be prompted for the server and database name.
