How to edit an RTF report

10560

Introduction

This article explains how to edit an RTF report

Symptom

Cause

Workaround

Resolution

Notes

  1. Make sure Encounter is in config mode. (See KB 10328 for details).
  2. Go to a patient's chart and find the report you wish to edit. Verify that the report it is an rtf report and not a Word report.
  3. Right click on the preview of the report. This will take you to the list of the report's display script commands. If there is not a local copy of the Display Script, create one.
  4. Edit Display Script
  5. See KB 10143 for descriptions of display script command options.
  6. Click OK to save changes.
  7. Make sure the Local Version of the Display Script is set to Active.
