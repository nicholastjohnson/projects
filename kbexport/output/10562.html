Creating Demo Users in EncounterPRO Version 5

10562

Introduction

 

When you run EncounterPRO version 5 against a Demo Database, many of the Demo users will already have a username/password registered in the EncounterPRO Authorization Database.  If you need to add or register a demo user, use the following rules to specify the user in EncounterPRO and to create the corresponding Yahoo! Email address.

 

Demo User Rules

 

Username = first name + last name no spaces all lowercase.

Password = encounterpro!

Password Question :  What is your pet’s name?

Password Answer: encounterpro

Email address:  epro.@yahoo.com

 

 

Yahoo Email Account Rules

 

First Name, Last Name, Gender:  Same as in Epro

Yahoo! ID:  epro.

Password: encounterpro!

Security Question:  What is your pet’s name?

Answer:  encounterpro

Birthday:  Any reasonable birthday

Zipcode:  30339

 

 

 Existing Demo Users should be listed in this KB article:  https://www.jmjtech.com/kb/Article.aspx?ArticleNumber=10563
