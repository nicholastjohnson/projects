Removing HP scanner

10564

Introduction

This article addresses problems that may be encountered while removing an HP scanner from the system. This information was taken in it's entirety from HP's web site.

Symptom

Attempts to Scan in EncounterPRO error-out because the application is unable to communicate with the HP drivers

Cause

The HP uninstall wizard fails to completely remove all program components which prevents EncounterPRO from  communicating with  the new scanner's software.

Workaround

HP Scanjet - Uninstall/Reinstall the HP Precisionscan, Pro, LT, or LTX Software in Safe Mode

      

 

 

This document provides instructions for uninstalling the HP Precisionscan, Pro, LT, and LTX and reinstalling it in Windows Safe Mode.

Windows NT 4.0 does not have a Safe Mode, however; instructions are provided for uninstalling and reinstalling the HP Precisionscan, Pro, LT, and LTX software in Windows NT 4.0.

Symptoms

The following are some of the symptoms that you may see and that would require using this document:

  • The Precisionscan software or scanner does not function correctly
  • Error messages appear when using the Precisionscan software
  • Error messages appear when trying to reinstall the Precisionscan software

Before beginning

Print this document for easy reference by using the Printable Version link in the right corner of this document. The document will not be available after the computer system has been turned off and restarted in Safe Mode.

Step 1: Disconnect the scanner from the computer system

  1. Shut down the computer system.
  2. Disconnect the scannerâ€™s power cable.
  3. Disconnect the interface cable (USB, parallel, or SCSI).
  4. Turn on the computer system.

Step 2: Uninstall the software

For Windows 95, 98, NT 4.0, and Me

1.       Click Start, Settings, then Control Panel. The Control Panel will open.

2.       Double-click Add/Remove Programs. The Add/Remove Programs Properties window will appear.

3.       Click Precisionscan, Pro, LT, or LTX to highlight it, then click Add/Remove. The Confirm File Deletion window appears. Click Yes.

4.       Click Yes to All on the shared files warning screen.

For Windows 2000 Pro

1.       Click Start, Settings, then Control Panel. The Control Panel window will open.

2.       Double-click Add/Remove Programs. The Add/Remove Programs Properties window will open.

3.       Click Precisionscan, Pro, LT, or LTX to highlight it, then click Change/Remove. The UninstallShield Wizardwindow will open.

4.       Click Yes to start the UninstallShield Wizard.

5.       Follow the screen prompts. Click Yes to All on the shared files warning screen.

For Microsoft Windows XP

1.       Click Start, then Control Panel. The Control Panel window will open.

2.       Double-click Add/Remove Programs. The Add/Remove Programs Properties window will open.

3.       Click Precisionscan, Pro, LT, or LTX to highlight it, then click Change/Remove. The UninstallShield Wizard window will open.

4.       Click Yes to start the UninstallShield Wizard.

5.       Follow the screen prompts. Click Yes to All on the shared files warning screen.

Step 3: Delete all remaining installation files

1.       Open the file search window.

o        In Windows 95, 98, and NT 4.0, click Start, Find, then Files or Folders. The Find: All Files window will open.

o        In Windows Me, 2000 Pro and XP, click Start, Search, then For Files or Folders. The Search Results window will open.

2.       In the Named field type: Scanjet*

3.       In the Look in field, select (C:). Make sure there is a check mark in the Include subfolders box.
