all folder content report is not printing attachments from all folders

10567

Introduction

this article explains how to print attachmetns from all folders from encounterpro.

Symptom

clicking "all folder content report" is not printing attachments from all folders.

Cause

Workaround

Resolution

Instead of using "All folder content" (there are some technical difficulties to fix this report) there is an alternate introducted with mod level 126 and higher versions. go to patients demographic screen and use "Char Export" button. this exports all the attachments from all folders into a single directory. then select all the files from the exported directory and print them.

in case if you do not see "Chart Export" button and the customer is in Mod Level 126 then run the hotfix 129 to create this "Chart Export" button to the demographics screen.
