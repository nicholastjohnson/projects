EPIE Message Monitor

10574

Introduction

This article explains some details about the EPIE Message Monitor service.

What it does:
Based on the results of this query (msg_Get_EpIE_Messages), this service moves files from one place to another. Each file has a corresponding message. This service sets that message's status to Processing.

Where it resides:
The service is currently running on the EIE server.
The source code can be found here in SourceSafe:
