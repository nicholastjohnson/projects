How to share folders with Unix

10575

Introduction

Some EncounterPRO Practice Management Integrations require EncounterPRO to share files with a Unix computer.  There are a number of products which make this possible by either sharing folders on the Unix computer which can be accessed from Windows, or by sharing folders on the Windows computer which can be accessed by Unix.  Any of these solutions will work and the system manager/integrator simply needs to insure that the appropriate software is installed and configured properly, and that the folders needed for the integration are shared properly and have the appropriate permissions set.

Our preferred product for sharing folders with a Unix computer is NFS Server from XLink Technologies, Inc.

This product can be purchased from here:  http://www.xlink.com

The latest installation file can probably be found on the JMJ FTP Site under EP_SupportingInstallers\Xlink, however it will require a serial number and password when it is installed that can only be acquired by purchasing a license.

Instructions for configuring NFS Server can be found here:  http://www.xlink.com/S-server2k.html  See the section labeled "Export drives and folders for NFS sharing"

Other support options, including free phone or email technical support can be found here:  http://www.xlink.com/support.htm
