Changing Menu on SOAP Chart Summary Rev tab

10583

Introduction

The Rev or Summary (Sigma) tab on the SOAP screen has a menu that you can update, but the menu name doesn't appear even when you are in Configuration mode.

The menu is controlled by a Preference: User-Defined > Special Favorite Child/Adult Chart Menu Summary.

Symptom

Cause

Workaround

1. Locate the Preference in the preferences list. There is a preference for Child and another for Adult. You should probably change both.

2. Select the Preference > Special Favorite Child/Adult Chart Menu Summary  and note the menu name. (Generally this will be: Standard Chart Menu Summary Screen Default and Standard Chart Menu Summary Screen Default Child) You could change the entire menu here at the preference, but if you just want to add/remove from the current menu you will need to do that via the Edit button.

3. BE SURE THAT YOU CHECK THE HIGHER PREFERENCE LEVELS! If there is a preference set at the Office, Specialty, or User level those will override the Global preference. Be sure you are editing the menu at the highest preference level. Highest level is User, lowest level is Global.

4. Select the Edit button under the menu name on the preference. Add/remove buttons as needed.

5. Restart EncounterPRO.
