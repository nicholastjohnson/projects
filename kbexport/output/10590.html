Order a treatment with a date that is older than encounter date

10590

Introduction

this article helps how to manually select a date and order a new treatment on closed encounters.

Symptom

Cause

Workaround

Resolution

Notes

a popup window will show up if the user is ordering a treatment on closed encounter. please check to make sure the treament mode is set to "History" (supposed to be as default on closed encounters) so that this pop up will show.

if the doc wants to order a lab that is done in the past as part of the current encounter then it can be done by adding a custom menu to their Other Items in Soap by setting the past_treatmnet atribute to true. now when they click this "Past Labs" custom menu it will bring a popup and allow them to log labs ordered in older dates.
