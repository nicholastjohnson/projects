Reopen treatments

10591

Introduction

Must have mod level 126.
After running hotfix 134, the buttons on the dashboard need to be created.

Go to Configuration, Treatment Type, pick a type and select Edit
Go to the Services tab, select Add Service, New Note/New Progress Note
Click on the new service and pick Edit Service
Change Button Title to ReOpen
Click on the new service again and choose Configure
Context Object = Treatment
Special Note Type = ReOpen
Other Note Title or Key = ReOpen
Note text - leave blank if you want them to be able to put in a reason why it's been reopened.  If they don't want the extra click, put in "." or "ReOpened" or anything.
