Error Posting Lab Results

10594

Introduction

Symptom

When posting a lab result from the Scan/Post screen, an error is reported to the event viewer like this:

Cannot insert the value NULL into column 'user_id', table 'cpr_126.dbo.p_Observation_Comment'; column does not allow nulls. INSERT fails.

 

Cause

This error is caused by a missing <ObservedBy> element in the XML file.

Workaround

Hotfix 141 adds logic to the stored procedure which posts results to use the observed_by value from a previous result for the same lab, if there is one.

Resolution

This issue will be resolved in EncounterPRO version 5.0.10
