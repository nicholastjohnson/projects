How to automatically print the contents of all the files in different subfolders that are exported thru CHART EXPORT service

10598

Introduction

this article explains how to automatically print the contents of all the files in different subfolders that are exported thru CHART EXPORT service

Symptom

chart export services exports all the documents to folders but they are not automatically printed. user needs to go and open each file and the request for printing.

Cause

Workaround

We tried adding printing to that service but we ran into too many problems trying to print attachments and with a bug in MS Word that caused an error message when printing the RTF documents. At this time the user is required to browse into the directory into which the files were exported and manually print the desired reports/attachments.

If the user wants to easily print all or multiple files, we have tested a 3rd party solution and find it useful, though we do not represent that it will print all attachments correctly. The product is called "Batch & Print Pro" from Traction Software. Their web site is here:

http://www.traction-software.co.uk/batchprint/index.html

Once installed the user can drag entire directories into the program window and it will print all the files it finds in the same order they appear in the directory.
