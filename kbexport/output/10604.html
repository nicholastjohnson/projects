Setting Coding for Insurance Carriers - Medicaid

10604

Introduction

The visit codes for Medicaid can be attached to the insurance carriers. If you need to change the code group for an insurance carrier, see the steps below.

Symptom

Cause

Workaround

This is a configuration setting. Go to Configuration > Insurance Carriers.

Locate the carrier and Edit.

Select the Authority Category of Medicaid and the Coding Algorithm of JMJ Medicaid Coding.
