Setting a Default Visit Billing Code for an Encounter Type

10606

Introduction

You can create an encounter type that will ALWAYS bill a certain visit code (for example, the Nurse Visit always bills a 99211).

The example below sets up a new encounter type to be used for vaccine clinic visits that the practice wants to bill as a level 2 office visit, 99212.

The drawing depicts the relationship between the tables that you will need to update if you are adding a new code group.

Notes

NOTE: If you are createing a new code group that follows the Office Visit Code rules, then you will be subject to those rules. So, even if you only enter a 99212 as the only code in your code group, if you don't have enough "points" for a level 2, you will only get a 99211 on your encounter billing. In the case of the vaccine only visit, I added a step to the workplan to enter the chief complaint and that put it up to a level 2.
