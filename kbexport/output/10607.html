Add a Print Button to Treatment Dashboard Screens

10607

Introduction

Some treatment dashboard screens already have or buttons. If you run across one that doesn't have one, there are 2 options:

 

Symptom

Cause

Workaround

1) Add a link to the display script for that dashboard.

2) Add a Service to the Treatment Type so that you get a button at the bottom of the screen. This is covered in another KB article. See Article #10033 - Adding Print button to Rx Dashboard.

Resolution

Adding to the Display script.

1) Display the treatment dashboard, right click to view the display script. Note the name of this Display Script! You will need it in a future step. If needed, create the Local Copy of the display script, then click Edit Display Script.

2) Add the following commands (lines) to the display script where you want the print button to appear. 

a) Highlight the first “print” command in the list and select “Insert New Command”.  This places the new command above the highlighted one. 

b) Select “General” at the top of the next screen.  Select “print” from the list.  A menu will appear prompting you to enter the text you want to appear.  Everything within the white typing area will appear on the screen.

c) For the Print command, you will need to add a Quick View menu. If there is not an appropriate Quick View menu already available, you can create one. (This step is needed so that you can attach the Display Script command to the "menu" button.)

d) Copy one of the Quick View menus (pick one that just has 1 button, preferably for a report). Give the new menu a descriptive name (you will need to remember this name!).

e) The menu config screen appears. Delete any buttons that already exist. Add  New button -- this will be a Service > Patient >EncounterPRO Report. When you configure the report button, do not select a report from the reports list. Keep selecting Next until you get to the Select Generic RTF report ID and select RTF Report. Select Next and then select the Display Script. This is the Display script for the dashboard screen (from step 1).

Select Next and save the menu. You will return to the Select Menu screen -- select the new Quick View menu you created in steps c and d. Continue to select Next and Finished until you are back at the display script edit screen.

4) Add a new command After your new print command. Select General, then Space. Add 5 spaces, then select OK. Select Finished to return to the dashboard display.

5) Refresh the dashboard (you may have to close it and re-open it). Select the link and preview the report.
