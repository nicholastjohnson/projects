ICD text descriptions are greater than the 80 character field limit

10612

Introduction

A small number of ICD9 text descriptions are greater than the 80 character field limit

Increasing the field character limit would be time consuming so need workaround

Note: When we move to ICD10, if there are many descriptions greater than 80, revisit Mark increasing field limit

Symptom

Only 80 characters show up as assessment (that is, some of the last few characters appear to have been cut off)

Cause

Field containing ICD description is limited to 80 characters

Workaround

There are a number of methods to reduce ICD description length without changing meaning

convert such phrases as "without" and "with" to "w/o" and "w/o" and "and" to "&"

delete unimportant small words such as "and" and "of"

delete punctuation chars such as , and .

But only do the above if it does not change the meaning!

If the above is insufficient consider the following

Contractions, abbreviations, deleting vowels, acronyms

But only if the meaning remains clear or can be recovered by clicking on the icon and going to the long description

Also, don't change the spelling or remove words that someone might want to search on

If you use an acronym (such as SIRS for systemic inflamatory response syndrome) go to www.pharma-lexicon.com to make sure that the acronym does not already stand for something else!
