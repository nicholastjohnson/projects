how to fix the lab results XML messages to be or not to be shown under scanned items

10614

Introduction

this article explains how to hide or unhide the lab results xml messages under scanned items tab

Symptom

a) customer complains that xml messages are showing under scanned items tab

b) jmj want to look up the original lab results message content to troubleshoot.

 

Cause

Workaround

Resolution

To hide lab results XML messages from display:

go to c_folder and make sure the lab data or lab reports folder is set to status = 'NA'

To unhide (show) the lab results xml messages..

go to c_folder and make sure the lab data or lab reports folder is set to status = 'OK'
