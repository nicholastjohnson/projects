EncounterPRO unable to connect connection broken

10615

Introduction

Symptom

The following error when starting EncounterPRO:

EncounterPRO was unable to connect to the SQL database.

Connection broken.

Cause

Workaround

Resolution

Run cliconfg.exe from Start\Run...

Change default network protocol to TCP/IP, click apply, and test.

If that doesn't work, try changing the default protocol to Named Pipes, click apply, and test.
