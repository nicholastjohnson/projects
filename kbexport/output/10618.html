Error adding new consultant

10618

Introduction

Mod level 127 adds functionality to unify the user list and the consultant list.  Under some circumstances, EncounterPRO is unable to create new consultants after upgrading to mod level 127.

Symptom

Adding a new consultant on the "Specialties and Consultants" screen results in an error message.  In the event view is an error or warning that contains text like this:

Cannot create user_id which is duplicate a consultant_id

 

Cause

Workaround

For databases on Mod Level 127, run Hotfix171.

Resolution

This issue is permanently resolved in Mod Level 128
