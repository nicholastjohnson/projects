EncounterPRO takes a long time to display a message folder

10621

Introduction

Symptom

On the messages tab, when the user selects a message folder other than "Inbox", EncounterPRO takes a long time to display the messages.

Cause

This condition is caused by the SQL Query Optimizer selecting a full index scan to find the messages.

Workaround

For Mod Level 126, run Hotfix 176.For Mod Level 127, run Hotfix 177.

Resolution

This issue is resolved in Mod Level 128.
