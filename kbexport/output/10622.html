How to reset the database objects

10622

Introduction

This article describes the steps to reset all of the database objects in an EncounterPRO database.

Symptom

Cause

Workaround

Resolution

For Mod Level 126

While the users are not using EncounterPRO:

 

  1. In a query window, run the “DB Scripts 126.sql” script.  The latest version of this script can be found here:

ftp://ftp.jmjtech.com/EP_Quick_Fixes/DB Scripts 126.sql

 

  1. In the DB Maintenance Utility, do the following:
    1. On the Updates tab, click “Check Now”
    2. On the Hotfix Scripts tab, click “Execute All”

 

 For Mod Level 127

 

While the users are not using EncounterPRO:

 

  1. In the DB Maintenance Utility, do the following:

a.       On the Updates tab, click “Check Now”

b.      On the Hotfix Scripts tab, click “Execute All”
