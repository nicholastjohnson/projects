Changing the report called from the Encounter Sigma Button

10625

Introduction

In several places there is a sigma (S) button which displays an encounter summary report.  The actual report called from the button is configurable.

Symptom

Cause

Workaround

If the sigma button is on the SOAP screen, follow the instructions in KB Article 10583.

For all other sigma buttons, the report called is configured in Configuration/Preferences->General Preferences: Default Encounter Summary Report.

If the Default Encounter Summary Report preference does not exist, run Hotfix 184.
