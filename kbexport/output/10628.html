How to configure IE 7 to use the CMA

10628

Introduction

IE 7 requires some config settings to use some features of the CMA such as opening an cust issue outlook form.

Symptom


Cause


Workaround


Resolution


Go to Tools/Internet Options and click on the Security tab

1) Make sure that https://www.jmjtech.com is one of your trusted sites.

 Select the "Trusted Sites" zone and click the "Sites"button.  Add "https://www.jmjtech.com" as a trusted site if it'snot already there.

2) Set options relating to ActiveX controls

Click the “Custom level..” button

Scroll down to the option labeled "Download unsigned ActiveX controls"
Set the option to "Enable"

Find the option labeled "Initialize and script ActiveX controls not marked as safe for scripting" (should be the next option)
Set the option to "Enable"
