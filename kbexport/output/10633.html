To transfer adatabase from one collation in SQL Server 2000 to a different collation in SQLServer 2000, follow these steps:

  1. Backup the source database.
  2. Createa new empty database on the destination server with the appropriate collation
  3. Generatescripts for all the objects in the source database
    1. Donot include the indexes, the triggers, the primary keys, the foreign keys, thedefault settings, and the constraints
    2. Makesure that you enable the “Only script 7.0 compatible features”
  4. Runthe scripts from step 3 on the destination database to create the objects withthe destination database collation
  5. UseDTS to transfer only the data fromthe source database.
  6. Afterthe data is successfully transferred, generate scripts for all the constraints,foreign keys, primary keys, and indexes from the source database.
  7. Runthe scripts from step 6 on the destination database.
