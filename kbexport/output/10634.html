Daily Maintenance Job Fails

10634

Introduction


Symptom

The daily maintenance job fails with an error message like this:

NT AUTHORITY\SYSTEM. Cannot insert the value NULL into column'last_hit', table '@listitems'; column does not allow nulls. INSERTfails. [SQLSTATE 23000] (Error 515)  The statement has beenterminated. [SQLSTATE 01000] (Error 3621).  The step failed

Cause

This can be caused when a freetext entry in the u_Top_20 table has a null value for the last_hit column.

Workaround

Run Hotfix 204

Resolution

This issue is resolved in Mod Level 128
