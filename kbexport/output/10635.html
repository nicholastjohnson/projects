Replacing an old v2 allergy with an active one

10635

Introduction

Symptom

Many version 2 drug allergies that get migrated to version 4 don't come across with the drugs mapped to them, so it will say that a patient is allergic to Penicillins, but no alert appears when penicillin is ordered. Do an Access search for patient allergy assessments linked to assessments which have no drugs in c_drug_allergy. For old drug allergies that have been ordered for many patients, write down the old assessment_id and find the id for "OK" statused drug allergies, and map the old one into the new one using the query below.

Cause

Workaround
