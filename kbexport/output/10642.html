How to turn on licensing at a customer site

10642

Introduction

Symptom

Cause

Workaround

Prerequisites:
Mod Level 127
Hotfix 216

If the program LicenseManager.exe is not in the EncounterPRO program directory, then download it from the FTP site and unzip it into the EncounterPRO program directory.  It will be in the FPT root folder called "LicenseManager"

Run the LicenseManager.exe program to manually license users and to turn on and off licensing.

Turnon licensing on a specific date, run this, replacing ‘mm/dd/yyyy’ with thedesired date:
EXECsp_set_preference 'SYSTEM', 'Global', 'Global', 'Workflow Model Standard Date','mm/dd/yyyy'
GO
