If you're configuring the attributes of the Refill service,whether from a workplan or a menu button, and it's saying "No ParametersDefined", click on the tiny little Edit button.

 

Add New param

            Param Class= Text Box

            Param Title= Default Refill Comment

            Attribute 1= refill_comment

            Req’d = N

Add New Param

            Param Class= Text Box

            Param Title= Default Print Attribute

            Attribute 1= print_when

            Req’d = N

 

Click Update, then OK

 

Possible values for the Default Refill Comment are “Refillfaxed to patient’s pharmacy” and “Refill called in to pharmacy number on file”

The three values that are accepted for the Default PrintAttribute are NOW CHECKOUT and DONT (noticethere’s no ‘ in DONT)
