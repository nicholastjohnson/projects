Un-cancelling leaves CC and comments still missing

10644

Introduction

Issue example: Evidently, missing information happens with the "Uncancel Encounter" option. The office manager called to say that she had uncancelled the encounter, but the CC was missing. Furthermore, the comments in the "Remarkable Addn'l History" were also not restored. I suspect that this would be true of free form text found in the physical as well.

Symptom

Cause

Hotfixes were run out of order.

Workaround

To fix the encounter, find the encounter_id, then go to p_observation_comment, filter for that encounter_id, and set the current_flags to Y

Resolution

Run ALL HOTFIXES in order so that this doesn't happen again.
