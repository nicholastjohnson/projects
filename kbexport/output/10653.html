Setting up Signature Capture in v2

10653

Introduction

Symptom

You need to tell Encounterpro that you have a signature pad avaiable on that computer.

Cause

Workaround

Resolution

To tell a computer to look for a signature pad, you need to perform the following steps while logged into the computer in question.Go to Configuration

Preferences

Change the Level to System

Leave the Computer setting where it is

Change the Preference Type to Computer

Configuration

Click on New Preference and select Signature Pad

Available and enter True and click OK

Click on New Preference and select Signature

Capture Software and enter CIC and click OK

Click Exit
