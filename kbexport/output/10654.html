Database Connectivity Issues

10654

Introduction

MDAC (Microsoft Data Access Components) is the set of software elements used by many programs, including EncounterPRO, to find and connect to their database. If the EncounterPRO client computer is unable to connect to the EncounterPRO database, this could be caused by problems with the MDAC software on the client.

Symptom

Any error message on the client that refers to problems with the SQL Server database or problems connecting to the database may be an MDAC problem. This includes error messages like these:

u_sqlca - (check()) SQL ERROR = (8114) SQLSTATE = 22018Microsoft OLE DB Provider for SQL ServerError converting data type varchar to int.

****

 u_common_thread - (get_adodb()) Error opening ADODB Connection (Error calling external object function open at line 20 in function get_adodb of object u_common_thread.)

****

Cause

Workaround

Resolution

Reinstall MDAC.  The latest versions can be found here:

http://msdn2.microsoft.com/en-us/data/aa937730.aspx
