Error opening chart

10655

Introduction

Symptom

When the user tries to open a chart, they get a message asking them to "Try again now" or "Try again later".  In the event viewer, an error message like this appears:


Error accessing external object property fontsize at line 25 in function initialize of object u_richtextedit_tx.

Cause

EncounterPRO uses several ActiveX controls to collect or display information to the user.  In a terminal server environment, ActiveX controls can be installed incorrectly or can become "broken" after installation.  In either case the terminal server might be left in a state where certain screens in EncounterPRO will not display correctly unless the windows user has Local Admin or Domain Admin privileges.

Workaround

A quick workaround is to put the EncounterPRO windows users in a group and grant that group Local Admin privileges on the terminal server.  This should only be treated as a temporary solution as it may consitute a security threat.

Resolution

Uninstall and resinstall EncounterPRO.  Make sure to use an administrator account to do this, and make sure to go through Control Panel - Add/Remove Programs.
