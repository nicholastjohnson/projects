Prescription Preferences

10656

Introduction

EncounterPRO has several preferences which control the display of the prescription report.

Symptom

Cause

Workaround

Resolution

 

 All preferences have the preference_type = "PREFERENCES" (General Preferences) and must be set at the Global level.

 

preference_id

Description in Epro Preferences

Usage

rx_always_use_signature_stamp

Custom Rx Use Signature On File Always

Select "Yes" to always use the signature on file, even if a signature is captured for the prescription.  Default is "No".

rx_gravityprompt

Custom Rx Signature Prompt

Text for authorization and verification of prescription.

rx_show_number_in_text

000Standard Rx Display Dispense Amount Textually

Select "Yes" to display dispense amounts both textually and numerically (e.g. fifty three (53)).  Select "No" to display dispense amounts only numerically.  Default is "No".

rx_use_signature_stamp

Custom Rx Use Signature On File

Select "Yes" to use the signature on file for a prescription where no signature was captured for that prescription.  Default is "No".

rx_include_generic_name

Display drug generic names

Select "Yes" to display drug generic names

rx_show_administer_frequency

N/A

Select “Code” to show only the code (e.g. BID).  Select “Description” to show only the description (e.g. Twice per Day).  Select “Code and Description” to show both.  Default is “Code”.

rx_dispense_amount_asterisks
RX Add Asterisks to Dispense Amount
Select "Yes" to display an asterisk (*) immediately before and after a dispense amount.  Default is "No". (v5 only)
