Creating new menu fails

10660

Introduction

Symptom

Creating a new menu or making a local copy of an existing menu fails with an error like this:Cannot insert the value NULL into column 'id', table 'cpr_50.dbo.c_Menu_Item'; column does not allow nulls. INSERT fails.

Cause

Workaround

Hotfix 239 fixes this problem for mod levels 120-131.

Resolution

This problem is fixed in mod level 132.
