Adding a One-Touch Reviewed button to treatment dashboard

10661

Introduction

Symptom

See Article #10641 - Preparation and configuration  to add a "Reviewed" link at the top of the dashboard screen if this is what practice prefers.

Cause

Workaround

Resolution

Go to Configuration --> Treatment Types, select the treatment type (lab, for example).

Select the tab labeled "Services" on the Treatment Type Definition screen.

Click the Add Service button.  Context Object is Treatment; Service is New Note.

Click on the service you just added, which will be at the bottom of the list of services on the Treatment Type Definition Screen.  Select Edit Service and change the button title to "One-Touch Reviewed".  Click OK. 

Click the service again and select Configure Service. 

Set the Context Object to Treatment

Standard Note Type:  Leave blank

Special Note Type:  Reviewed

Preconfigured Treatment Titles or Keys:  Leave blank

Other Note Title or Key:  Lab

Note Text:  Results and status reviewed

Click Finished.

 

(the following is an excerpt from KB article # 10641)

Who reviewed the lab does not appear after clicking

To allow the dashboard to show which user reviewed the lab perform the following:

     Right click anywhere inside a lab dashboard > create a local copy if necessary then

     click 'Edit Display Script' in the commands menu to the left and scroll for the 'Treatment

     Reviewed By' or 'Treatment Progress' that, when highlighted, states

     "progress_type=Reviewed" and enable one or both of those commands. > click "ok" > click

     "finish". (See Disabling the time or date command from appearing after clicking

     "reviewed" to understand differences of the two commands.)

 

Disabling the time or date from appearing after clicking "reviewed"  

After a user clicks the button,  the format may appear at bottom of lab dashboard as shown below:

14:50

Reviewed By Support

Results and status reviewed.

5/22/2007

Support

Lab

Results and status reviewed.

The top line lists the time lab was reviewed, which is the 'Treatment Reviewed By' command and the bottom line lists the reviewed date, which is one of the 'Treatment Progress' commands listed in the commands menu.

 

If practice wishes to have the time removed do the following:

     Right click on lab dashboard > edit dashboard (with local copy active) > then scroll down

     the commands menu to the left and find 'Treatment reviewed by' and disable that command

     > click "ok" > click "finish" and the review will now appear as follows:

5/22/2007

Support

Lab

Results and status reviewed.
