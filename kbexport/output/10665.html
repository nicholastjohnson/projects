Introduction

The DevIssues installation program can be found on the FTP Site under EP_Upgrades\Production\DevIssues. Generally it's a good idea to install the latest version.Devissues shares some components with EncounterPRO so if DevIssues isn't working, try installing the latest version of the EncounterPRO client.
