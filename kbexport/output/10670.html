Scan post button doesn't work

10670

Introduction

Symptom

When you click on Scan/Post, nothing happens (but there is an error message:EproLibNET version: 4.106.8.0w_scan_main - (open) Error creating posting listsEproLibNET version: 4.106.8.0Violation of PRIMARY KEY constraint 'PK_c_Folder'. Cannot insert duplicate key in object 'c_Folder'.

Cause

The status of folder "Items to be Posted" in c_folder is set to NA

Workaround

Resolution

Set the status of "Items to be posted" to OK

Notes
