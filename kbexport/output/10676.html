How to create a KB Article with Images

10676

Introduction

How to create a KB article, which includes any images you want to appear in your article.

Symptom

Boring looking KB articles with NO Pictures!

Cause

We are unable to paste images here in these text boxes.

Workaround

Do not place the image in the knowledge base article at all. Let someone else get a mental visual.

Resolution

  1. Click on Create New Article link
  2. Save form and get the kb article number
  3. Create a folder here: \\Wwwserver\jmjprod\KB\images naming the folder your kb article number.
  4. Place the file to be displayed into the folder.
  5. Add the html syntax to have ur image appear.

You can display any file! However, if u want to use the standard types, save ur file as either bmp, gif, jpg, png and .tiff file format types! =)

Notes

You will need to know a bit of html to pull this one off.
