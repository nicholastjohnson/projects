TX4OLE Insufficient Memory Error

10677

Introduction

This insufficient memory problem exists across applications and is not a direct result of using EncounterPro.

Symptom

Cause

It seems to appear when you modify the Tx Text Control to display tables or reports with RTF, and/or have citrix applications running on your machine.

Workaround

Resolution

Click OK and continue working.


The vendor “The Imaging Source Europe GmbH” www.theimagingsource.com; is aware of this issue and has not posted a resolution for it at the time of this writing.
