Importing signature into user's profile

10678

Introduction

A new provider has joined the office and needs her signature imported into her user profile for EncounterPRO so that it will appear on reports/prescriptions

Symptom

Cause

Workaround

Resolution

Email the practice a copy of the signature sheet (found in common on labserv2003\Support\Signature Sheet) if they do not already have it.  Have the new provider sign within the designated area and fax the sheet back to us.  To see the incoming faxes on the server, go to Start --> Run, paste in "\\jmjserver\Incoming_faxes".  Find the fax.  Box in the signature by clicking th mouse, holding it down and moving it to enclose the area.  Leave as little white space around it as possible.  Right click and select copy image.  Open Paint and paste the image in paint.  Adjust it if needed.

Now you will copy the image to the ftp site so that you can copy it to the practice's server.  Go to Start--> Run, paste in "ftp.jmjtech.com".  Open the Uploads folder, find the appropriate practice (if a folder for the practice does not exist, create one). Copy the image into the practice's flder under Uploads.  Now go to the practice's server.  Open the ftp site from their server and copy the signature onto the desktop so that it is very easy to find. 

Open EncounterPRO, go to Configuration --> Users, select the appropriate user.  Select the user and select Edit.  Select the Signature Stamp button at the bottom of the screen.  Select import Bitmap file.  Navigate to the desktop and select the signature you just copied there.  Click OK and the signature should now be in the profile.  Be sure to delete the signature from the desktop!
