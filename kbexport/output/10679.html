Refer Patient for Treatment billing flag buttons

10679

Introduction

Symptom

Question: The “Refer Patient for Treatment” screen’s billing buttons don’t automatically flip to No like they used to – Where do we set the default setting for these billing flags?

Cause

Answer(from Mark): Those buttons track the treatment instance’s values for bill_observation_collect and bill_observation_perform in p_Treatment_Item. Those flags are initialized from the same named fields in c_Treatment_Type when the treatment is ordered.
