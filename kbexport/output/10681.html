Notes

Sometime while activating billing interface the client prefers to release billing for one provider or two provider encounters before sending everyones. This is to make sure the billing interface accuratly sending the charge data to billing system. EncounterPRO 4.x and above does supports this by listing the providers the charges should be released in c_Domain table. There is no user interface but is considered as enhancement request for EPRO Server 8.x release. Here is how to setup..

To send billing only for selective providers, add a record in c_domain as below.

dbo_c_Domain
domain_iddomain_sequencedomain_itemdomain_item_descriptiondomain_item_bitmapsort_sequence
billing_provider1SUPPORTSupport

here the domain_item should be the user_id from c_user of the billable provider the charge should be released .

To release billing for everyone, remove the "billing_provider" records from c_Domain or add all the billable providers in the c_domain under billing_provider as explained above.
